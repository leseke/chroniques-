# UX

## Rôle

Le dossier **UX** regroupe toute la documentation relative à l’expérience utilisateur de Chroniques.

Il définit la manière dont le joueur interagit avec le jeu afin d’offrir une expérience intuitive, fluide, accessible et cohérente.

—

## Objectifs

- Concevoir une expérience utilisateur de qualité.
- Garantir une navigation intuitive.
- Améliorer le confort de jeu.
- Assurer l’accessibilité et la cohérence des interactions.

—

## Contenu

- Parcours utilisateur
- Ergonomie
- Navigation
- Menus
- HUD
- Interfaces
- Accessibilité
- Feedback visuels et sonores
- Tutoriels
- Tests utilisateurs

—

## Convention

Chaque document est identifié par un numéro unique.

Exemples :

- UX-001
- UX-002
- UX-003

—

## Dépendances

Le contenu du dossier UX respecte les décisions définies dans les documents MASTER, les spécifications de la GDB ainsi que la direction artistique du dossier ART.
