GDB-018B --- Les Lois

Version : 1.0 Statut : Officiel Type : Justice & Droit
⸻
OBJECTIF

Définir les principes fondamentaux des lois dans Chroniques.

Les lois constituent les règles officielles qui organisent la vie en 
société. Elles définissent les droits, les obligations et les limites 
des comportements acceptés afin d'assurer l'ordre, la sécurité et la 
cohésion des communautés.
⸻
PRINCIPE

Une loi est une règle reconnue par une autorité.

Son contenu reflète les valeurs, les besoins et l'histoire de la société 
qui l'a adoptée. Les lois évoluent avec le temps et peuvent être créées, 
modifiées, interprétées ou supprimées.
⸻
LES CATÉGORIES DE LOIS

Les lois peuvent notamment concerner :

les personnes ;
les biens ;
le commerce ;
les territoires ;
les institutions ;
l'environnement ;
les relations entre communautés.

Chaque société possède son propre corpus juridique.
⸻
ÉVOLUTION

Les lois évoluent grâce :

aux réformes ;
aux décisions judiciaires ;
aux changements politiques ;
aux transformations sociales ;
aux générations.

Elles peuvent renforcer ou transformer l'organisation d'une 
civilisation.
⸻
IMPACT

Les lois influencent :

les comportements ;
les institutions ;
les échanges ;
les conflits ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux lois devra :

refléter les valeurs de chaque société ;
évoluer avec les événements du monde ;
produire des conséquences cohérentes ;
interagir avec les autres systèmes ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle des lois un véritable cadre vivant de la 
société plutôt qu'une simple liste de règles ?

Si la réponse est non, elle devra être repensée.
