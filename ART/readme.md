# ART

## Rôle

Le dossier **ART** regroupe toute la documentation liée à la direction artistique de Chroniques.

Il définit l’identité visuelle du jeu, les références artistiques, les règles graphiques et les ressources nécessaires à la production.

—

## Objectifs

- Garantir une identité visuelle cohérente.
- Documenter les choix artistiques.
- Servir de référence pour la création des assets.
- Assurer une cohérence graphique sur l’ensemble du projet.

—

## Contenu

- Direction artistique
- Concept art
- Personnages
- Environnements
- Objets
- Interface utilisateur (UI)
- Palette de couleurs
- Typographie
- Icônes
- Effets visuels (VFX)

—

## Convention

Chaque document est identifié par un numéro unique.

Exemples :

- ART-001
- ART-002
- ART-003

—

## Dépendances

Le contenu du dossier ART respecte les décisions définies dans les documents MASTER, les spécifications de la GDB et les éléments narratifs du dossier LORE.
