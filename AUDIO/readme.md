# AUDIO

## Rôle

Le dossier **AUDIO** regroupe toute la documentation liée à l’identité sonore de Chroniques.

Il définit les musiques, les effets sonores, les ambiances, les voix et les règles de conception audio du jeu.

—

## Objectifs

- Garantir une identité sonore cohérente.
- Documenter les besoins audio.
- Organiser les ressources sonores.
- Servir de référence pour la production et l’intégration audio.

—

## Contenu

- Bande originale (OST)
- Musiques d’ambiance
- Effets sonores (SFX)
- Voix et dialogues
- Ambiances sonores
- Interface utilisateur (UI)
- Notifications
- Directives de mixage

—

## Convention

Chaque document est identifié par un numéro unique.

Exemples :

- AUDIO-001
- AUDIO-002
- AUDIO-003

—

## Dépendances

Le contenu du dossier AUDIO respecte les décisions définies dans les documents MASTER, les spécifications de la GDB ainsi que la direction artistique du dossier ART.
