GDB-001G --- La Civilisation

Version : 1.0
Statut : Officiel
Type : Principe Fondateur
⸻


OBJECTIF

Définir la philosophie de la civilisation partagée dans Chroniques.

Ce document décrit les principes reliant les joueurs entre eux.

Il ne définit pas les systèmes sociaux, les guildes ou le commerce.
⸻
UNE CIVILISATION COMMUNE

Tous les joueurs appartiennent à une même civilisation.

Ils ne partagent pas obligatoirement le même monde.

Ils partagent une histoire plus vaste qu'eux.
⸻
UN MONDE PERSONNEL

Chaque joueur possède un monde qui lui appartient.

Ses décisions façonnent ce monde.

Les autres joueurs ne modifient jamais directement son histoire sans son 
consentement.
⸻
UNE SOCIÉTÉ CONNECTÉE

La civilisation permet notamment :

	⁃	les échanges ;
	⁃	la coopération ;
	⁃	la transmission des connaissances ;
	⁃	la communication ;
	⁃	les rencontres ;
	⁃	les œuvres communes.

La connexion entre joueurs enrichit l'expérience sans retirer l'identité 
de leur monde personnel.
⸻
LE SENTIMENT D'APPARTENANCE

Chroniques cherche à créer le sentiment de faire partie d'une même 
civilisation.

Ce sentiment ne repose pas sur un monde unique.

Il repose sur les interactions, la mémoire commune et les liens créés 
entre les joueurs.
⸻
RESPECT DE L'AUTONOMIE

Chaque joueur reste maître de son aventure.

La coopération est un choix.

Jamais une obligation.
⸻
RÈGLES DE CONCEPTION

Toute mécanique multijoueur devra :

	1.	préserver le monde personnel ;
	2.	renforcer le sentiment de civilisation ;
	3.	respecter la liberté de chaque joueur ;
	4.	limiter la complexité technique inutile ;
	5.	enrichir les histoires sans les uniformiser.
⸻
CRITÈRE DE VALIDATION

Cette mécanique renforce-t-elle le sentiment d'appartenir à une 
civilisation tout en préservant l'indépendance du monde personnel ?

Si la réponse est non, elle devra être repensée.
