OBJECTIF

Définir les principes fondamentaux des expérimentations dans Chroniques.

Les expérimentations sont les procédures permettant de tester des 
hypothèses, de valider des théories et de produire de nouvelles 
connaissances. Elles constituent le lien entre les idées scientifiques 
et leur application concrète.
⸻
PRINCIPE

Une expérimentation consiste à observer les effets d'une action réalisée 
dans des conditions maîtrisées.

Elle doit être reproductible, documentée et analysée afin de distinguer 
les résultats fiables des erreurs, des coïncidences ou des 
interprétations incorrectes.
⸻
LES COMPOSANTES

Toute expérimentation comprend généralement :

une hypothèse à tester ;
un protocole expérimental ;
des variables contrôlées ;
des instruments de mesure ;
une collecte des données ;
une analyse des résultats ;
une conclusion.

Les expérimentations peuvent être répétées ou adaptées pour confirmer 
les observations.
⸻
ÉVOLUTION

Les expérimentations évoluent grâce :

aux progrès scientifiques ;
aux innovations technologiques ;
aux nouveaux équipements ;
aux collaborations entre chercheurs ;
aux découvertes antérieures.

Le niveau de précision augmente avec le développement des civilisations.
⸻
IMPACT

Les expérimentations influencent :

les découvertes scientifiques ;
les innovations ;
les technologies ;
la médecine ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux expérimentations devra :

récompenser la rigueur scientifique ;
accepter l'échec comme source d'apprentissage ;
produire des résultats crédibles ;
interagir avec l'ensemble des systèmes scientifiques ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle des expérimentations un véritable processus 
de découverte plutôt qu'une simple action automatique ?

Si la réponse est non, elle devra être repensée.
