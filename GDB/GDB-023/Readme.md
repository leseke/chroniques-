# GDB-023

> Les sciences et les technologies qui font progresser le monde.

---

# Présentation

La série **GDB-023** définit les mécanismes de recherche, de découverte et d'innovation qui permettent l'évolution scientifique et technologique de Chroniques.

---

# Objectif

La série GDB-023 répond à la question :

> **Comment les découvertes transforment-elles progressivement le monde ?**

---

# Cette série couvre

- les sciences ;
- la recherche ;
- les découvertes ;
- les inventions ;
- les expérimentations ;
- les innovations ;
- les applications ;
- la diffusion des connaissances ;
- le progrès ;
- la philosophie des sciences et des technologies.
