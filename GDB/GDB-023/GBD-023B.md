GDB-023B --- Les Disciplines Scientifiques

Version : 1.0
Statut : Officiel
Type : Sciences & Technologies
⸻


OBJECTIF

Définir les principes fondamentaux des disciplines scientifiques dans 
Chroniques.

Les disciplines scientifiques organisent les connaissances en domaines 
spécialisés afin de faciliter l'étude, la recherche et le développement 
des savoirs. Elles permettent aux civilisations de progresser dans des 
secteurs variés tout en favorisant les interactions entre les 
différentes branches de la science.
⸻
PRINCIPE

Une discipline scientifique est un domaine d'étude structuré.

Chaque discipline possède ses méthodes, ses concepts, ses outils et ses 
objectifs propres. Les découvertes réalisées dans une discipline peuvent 
enrichir ou transformer les autres.
⸻
LES PRINCIPALES DISCIPLINES

Les disciplines scientifiques peuvent notamment être :

les sciences naturelles ;
les sciences physiques ;
les sciences chimiques ;
les sciences biologiques ;
les sciences médicales ;
les sciences sociales ;
les sciences de l'ingénieur.

Leur nombre et leur niveau de spécialisation évoluent avec les 
civilisations.
⸻
ÉVOLUTION

Les disciplines évoluent grâce :

aux découvertes ;
aux recherches ;
aux échanges interdisciplinaires ;
aux innovations ;
aux générations.

De nouvelles disciplines peuvent apparaître tandis que d'autres 
fusionnent ou disparaissent.
⸻
IMPACT

Les disciplines scientifiques influencent :

les technologies ;
la médecine ;
l'économie ;
l'éducation ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux disciplines scientifiques devra :

permettre la spécialisation des savoirs ;
favoriser les interactions entre disciplines ;
évoluer avec les découvertes ;
interagir avec les autres systèmes ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle des disciplines scientifiques des domaines 
vivants capables d'évoluer et de s'enrichir mutuellement plutôt qu'une 
simple classification ?

Si la réponse est non, elle devra être repensée.
