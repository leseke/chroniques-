GDB-008B --- Les Saisons

Version : 1.0
Statut : Officiel
Type : Temps & Générations
⸻


OBJECTIF

Définir les principes fondamentaux des saisons dans Chroniques.

Les saisons rythment naturellement la vie du monde. Elles modifient les 
paysages, les activités, les ressources et les comportements sans jamais 
constituer une contrainte artificielle.
⸻
PRINCIPE

Chaque saison possède une identité propre.

Elle influence le monde de manière cohérente tout en laissant au joueur 
la liberté d'adapter son mode de vie.
⸻
UN MONDE EN ÉVOLUTION

Les saisons modifient notamment :

le climat ;
les cultures ;
la faune et la flore ;
les récoltes ;
les déplacements ;
les métiers ;
les traditions.

Le monde reste vivant grâce à ces cycles naturels.
⸻
DES OPPORTUNITÉS DIFFÉRENTES

Chaque saison ouvre de nouvelles possibilités.

Aucune n'est meilleure qu'une autre.

Le joueur adapte ses projets selon ses envies et les opportunités du 
moment.
⸻
CONSÉQUENCES

Les saisons influencent progressivement les habitants, les marchés, les 
ressources et les événements.

Ces changements restent prévisibles tout en laissant une place à 
l'imprévu.
⸻
IMPACT

Les saisons renforcent :

l'immersion ;
la diversité des expériences ;
la planification ;
les histoires émergentes ;
le sentiment d'un monde vivant.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux saisons devra :

respecter la cohérence écologique ;
créer de nouvelles opportunités plutôt que des contraintes ;
enrichir les activités existantes ;
évoluer naturellement avec le monde ;
renforcer l'identité de chaque période de l'année.
⸻
CRITÈRE DE VALIDATION

Cette mécanique utilise-t-elle les saisons pour enrichir l'expérience 
du joueur plutôt que pour limiter artificiellement ses possibilités ?

Si la réponse est non, elle devra être repensée.
