GDB-008E --- Le Vieillissement

Version : 1.0
Statut : Officiel
Type : Temps & Générations
⸻


OBJECTIF

Définir les principes fondamentaux du vieillissement dans Chroniques.

Le vieillissement représente l'évolution naturelle d'un individu au fil 
du temps. Il modifie progressivement ses capacités, ses priorités et son 
rôle dans le monde.
⸻
PRINCIPE

Vieillir n'est ni une récompense ni une punition.

C'est une transformation naturelle. Chaque période de la vie possède ses 
forces, ses limites et ses opportunités.
⸻
ÉVOLUTION

Le vieillissement agit progressivement sur :

les capacités physiques ;
les connaissances ;
la maîtrise ;
les responsabilités ;
les relations ;
les ambitions.
⸻
UNE NOUVELLE PLACE

Avec l'âge, un personnage peut :

transmettre son savoir ;
conseiller ;
former ;
gérer un patrimoine ;
adopter un nouveau rythme de vie.
⸻
IMPACT

Le vieillissement influence :

les familles ;
les métiers ;
la transmission ;
les communautés ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique devra :

évoluer progressivement ;
valoriser chaque période de la vie ;
créer de nouvelles opportunités ;
éviter les pénalités arbitraires ;
renforcer la continuité des générations.
⸻
CRITÈRE DE VALIDATION

Cette mécanique transforme-t-elle le vieillissement en évolution 
plutôt qu'en simple déclin ?

Si la réponse est non, elle devra être repensée.
⸻
