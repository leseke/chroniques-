GDB-008C --- Les Cycles de Vie

Version : 1.0
Statut : Officiel
Type : Temps & Générations
⸻


OBJECTIF

Définir les principes fondamentaux des cycles de vie dans Chroniques.

Chaque être vivant évolue au fil du temps. Les cycles de vie structurent 
naturellement les générations, les métiers, les relations et l'histoire 
du monde.
⸻
PRINCIPE

La vie est un processus.

Chaque individu traverse différentes étapes qui influencent 
progressivement ses capacités, ses responsabilités et ses ambitions.
⸻
LES GRANDES ÉTAPES

Le cycle de vie comprend notamment :

la naissance ;
l'enfance ;
l'adolescence ;
l'âge adulte ;
la maturité ;
la vieillesse ;
la fin de vie.

Chaque étape ouvre de nouvelles possibilités sans effacer totalement les 
précédentes.
⸻
UNE ÉVOLUTION CONTINUE

Les changements sont progressifs.

Les compétences, les relations, les connaissances et les responsabilités 
évoluent naturellement avec l'expérience accumulée.
⸻
IMPACT SUR LE MONDE

Les cycles de vie influencent :

les familles ;
les métiers ;
la transmission ;
l'économie ;
les communautés ;
les histoires émergentes.

Le monde évolue génération après génération.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux cycles de vie devra :

respecter une évolution progressive ;
valoriser chaque période de la vie ;
créer des opportunités propres à chaque étape ;
favoriser la transmission ;
renforcer la cohérence du monde.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle du sens au passage de la vie plutôt que 
de simplement faire avancer une chronologie ?

Si la réponse est non, elle devra être repensée.
⸻
