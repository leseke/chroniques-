GDB-008H --- Les Traditions

Version : 1.0
Statut : Officiel
Type : Temps & Générations
⸻


OBJECTIF

Définir les principes fondamentaux des traditions dans Chroniques.

Les traditions représentent les pratiques, les savoir-faire et les 
valeurs qui traversent le temps. Elles donnent une identité aux 
familles, aux communautés et aux cultures.
⸻
PRINCIPE

Une tradition est une pratique transmise d'une génération à une autre.

Elle évolue avec le temps et n'est jamais figée.
⸻
ORIGINE

Les traditions peuvent naître :

d'un événement marquant ;
d'un métier ;
d'une famille ;
d'une communauté ;
d'une découverte ;
d'une croyance ;
d'un personnage influent.
⸻
ÉVOLUTION

Chaque génération peut :

préserver une tradition ;
l'adapter ;
l'enrichir ;
l'abandonner ;
en créer une nouvelle.

Les traditions vivent au rythme du monde.
⸻
IMPACT

Les traditions influencent :

l'identité des familles ;
les communautés ;
les relations ;
les métiers ;
les histoires émergentes.

Elles participent à la mémoire collective.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux traditions devra :

évoluer naturellement ;
respecter la liberté des générations ;
renforcer l'identité culturelle ;
créer des conséquences à long terme ;
enrichir le monde vivant.
⸻
CRITÈRE DE VALIDATION

Cette mécanique permet-elle aux traditions d'évoluer avec le monde 
plutôt que de rester figées ?

Si la réponse est non, elle devra être repensée.
⸻
