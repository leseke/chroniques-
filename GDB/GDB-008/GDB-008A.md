GDB-008A --- Le Temps

Version : 1.0
Statut : Officiel
Type : Temps & Générations
⸻


OBJECTIF

Définir les principes fondamentaux du temps dans Chroniques.

Le temps est une composante vivante du monde. Il transforme les lieux, 
les habitants, les projets et les générations.
⸻
PRINCIPE

Le temps n'est pas un simple compteur.

Il représente l'évolution naturelle du monde et donne du sens aux choix 
du joueur.
⸻
UN TEMPS VIVANT

Le passage du temps influence notamment :

les saisons ;
les cultures ;
les relations ;
les métiers ;
les constructions ;
les ressources ;
les habitants.

Le monde évolue même lorsque le joueur change de priorité.
⸻
LE TEMPS COMME RESSOURCE

Le temps possède une valeur.

Choisir une activité signifie renoncer temporairement à une autre.

Cette contrainte crée des décisions intéressantes sans imposer de 
pression artificielle.
⸻
ÉVOLUTION

Avec le temps :

les projets avancent ;
les enfants grandissent ;
les habitants vieillissent ;
les traditions évoluent ;
les souvenirs deviennent un héritage.
⸻
IMPACT

Le temps renforce :

les histoires émergentes ;
la transmission ;
l'attachement ;
les conséquences ;
la crédibilité du monde.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée au temps devra :

produire des conséquences naturelles ;
respecter le rythme du joueur ;
éviter les attentes artificielles ;
enrichir l'évolution du monde ;
donner du sens aux générations.
⸻
CRITÈRE DE VALIDATION

Cette mécanique utilise-t-elle le temps pour enrichir le monde plutôt 
que pour ralentir artificiellement le joueur ?

Si la réponse est non, elle devra être repensée.
⸻
