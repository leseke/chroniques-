GDB-008I --- La Mémoire du Monde

Version : 1.0
Statut : Officiel
Type : Temps & Générations
⸻


OBJECTIF

Définir les principes fondamentaux de la mémoire du monde dans 
Chroniques.

La mémoire du monde conserve les traces des générations successives et 
permet aux actions passées d'influencer durablement l'avenir.
⸻
PRINCIPE

Le monde se souvient.

Les habitants, les lieux, les familles et les institutions conservent 
une partie de leur histoire.

Les conséquences importantes ne disparaissent pas simplement avec le 
temps.
⸻
LES FORMES DE MÉMOIRE

La mémoire du monde peut prendre la forme de :

monuments ;
archives ;
traditions ;
récits ;
réputation ;
patrimoine ;
paysages transformés ;
lignées familiales.

Chaque trace raconte une partie de l'histoire.
⸻
UNE MÉMOIRE ÉVOLUTIVE

La mémoire n'est jamais parfaite.

Certains événements sont oubliés.

D'autres sont déformés, redécouverts ou réinterprétés par les 
générations suivantes.
⸻
IMPACT

La mémoire influence :

les cultures ;
les relations ;
les décisions ;
les héritages ;
les histoires émergentes.

Elle renforce la profondeur historique du monde.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée à la mémoire devra :

conserver les conséquences importantes ;
accepter l'évolution des récits ;
renforcer l'identité du monde ;
créer des liens entre les générations ;
enrichir la narration émergente.
⸻
CRITÈRE DE VALIDATION

Cette mécanique permet-elle au monde de conserver une véritable 
histoire plutôt qu'un simple historique d'événements ?

Si la réponse est non, elle devra être repensée.
⸻
