# GDB-020

> Les organisations structurées du monde de Chroniques.

## Présentation

La série **GDB-020** définit les différents types d'organisations présents dans Chroniques et leur rôle dans la société.

## Objectif

Répondre à la question :

> Comment les individus se regroupent-ils pour poursuivre des objectifs communs ?

## Cette série couvre

- les organisations ;
- les guildes ;
- les ordres ;
- les confréries ;
- les compagnies ;
- les corporations ;
- les syndicats ;
- les associations ;
- les fondations ;
- la philosophie des organisations.
