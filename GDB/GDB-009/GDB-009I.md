GDB-009I --- Les Conflits

Version : 1.0
Statut : Officiel
Type : Société & Civilisations
⸻


OBJECTIF

Définir les principes fondamentaux des conflits dans Chroniques.

Les conflits représentent les tensions naturelles entre individus, 
familles, communautés, institutions ou civilisations. Ils participent à 
l'évolution du monde sans en constituer la finalité.
⸻
PRINCIPE

Un conflit naît d'une divergence d'intérêts, de valeurs, de ressources 
ou d'objectifs.

Il peut être résolu, transformé, aggravé ou disparaître avec le temps.

Aucun conflit n'est permanent par nature.
⸻
LES FORMES DE CONFLITS

Les conflits peuvent être :

personnels ;
familiaux ;
économiques ;
politiques ;
culturels ;
territoriaux ;
idéologiques.

Ils ne se limitent jamais à la violence.
⸻
ÉVOLUTION

Les conflits évoluent selon :

les décisions des acteurs ;
les négociations ;
les événements ;
les générations ;
les changements de société.

Ils produisent des conséquences durables.
⸻
IMPACT

Les conflits influencent :

les relations ;
les lois ;
les communautés ;
les institutions ;
les échanges ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux conflits devra :

avoir une origine crédible ;
proposer plusieurs moyens de résolution ;
produire des conséquences cohérentes ;
évoluer naturellement avec le monde ;
enrichir la profondeur sociale de Chroniques.
⸻
CRITÈRE DE VALIDATION

Cette mécanique traite-t-elle les conflits comme des phénomènes 
vivants plutôt que comme de simples combats ?

Si la réponse est non, elle devra être repensée.
