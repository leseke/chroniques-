# GDB-009

> Les structures sociales et les civilisations.

---

# Présentation

La série **GDB-009** décrit la manière dont les sociétés s'organisent et évoluent.

Elle couvre les communautés, les familles, les cultures, les institutions, les lois et les conflits qui façonnent les civilisations présentes dans Chroniques.

---

# Objectif

La série GDB-009 répond à la question :

> **Comment les sociétés se construisent-elles et évoluent-elles ?**

---

# Cette série couvre

- les sociétés ;
- les communautés ;
- les familles ;
- les cultures ;
- les valeurs ;
- les lois ;
- les institutions ;
- les gouvernances ;
- les conflits ;
- la philosophie des civilisations.

---

# Cette série ne couvre pas

- les systèmes économiques ;
- les mécaniques techniques ;
- les aspects purement géographiques.
