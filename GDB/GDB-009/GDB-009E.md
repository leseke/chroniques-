GDB-009E --- Les Valeurs

Version : 1.0
Statut : Officiel
Type : Société & Civilisations
⸻


OBJECTIF

Définir les principes fondamentaux des valeurs dans Chroniques.

Les valeurs représentent les principes qui guident les choix des 
individus, des familles, des communautés et des civilisations. Elles 
façonnent durablement les comportements et les relations.
⸻
PRINCIPE

Les valeurs évoluent avec le temps.

Elles sont influencées par l'histoire, la culture, les traditions, les 
expériences et les générations.

Aucune valeur n'est universellement supérieure.
⸻
LES SOURCES DES VALEURS

Les valeurs peuvent naître de :

la famille ;
la culture ;
la religion ;
l'éducation ;
les événements historiques ;
les expériences personnelles ;
les relations.
⸻
ÉVOLUTION

Les valeurs peuvent :

se renforcer ;
évoluer ;
entrer en conflit ;
être transmises ;
disparaître progressivement.

Le monde change avec elles.
⸻
IMPACT

Les valeurs influencent :

les décisions ;
les lois ;
les traditions ;
les alliances ;
les conflits ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux valeurs devra :

respecter leur diversité ;
permettre leur évolution ;
éviter toute hiérarchie arbitraire ;
influencer naturellement les comportements ;
enrichir la profondeur sociale du monde.
⸻
CRITÈRE DE VALIDATION

Cette mécanique permet-elle aux valeurs d'influencer durablement la 
société sans enfermer les individus dans un comportement unique ?

Si la réponse est non, elle devra être repensée.
⸻
