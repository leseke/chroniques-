GDB-009B --- Les Communautés

Version : 1.0
Statut : Officiel
Type : Société & Civilisations
⸻


OBJECTIF

Définir les principes fondamentaux des communautés dans Chroniques.

Les communautés regroupent des individus partageant des intérêts, des 
valeurs, un territoire ou un objectif commun. Elles constituent 
l'échelle sociale la plus vivante du monde.
⸻
PRINCIPE

Une communauté est une organisation naturelle d'individus.

Elle se construit progressivement grâce aux relations, à la confiance, 
aux traditions et aux expériences partagées.
⸻
LES FONDEMENTS D'UNE COMMUNAUTÉ

Une communauté peut être unie par :

un territoire ;
une activité ;
une culture ;
une famille ;
une croyance ;
un objectif commun ;
une histoire partagée.

Ces liens évoluent avec le temps.
⸻
ÉVOLUTION

Les communautés peuvent :

grandir ;
se diviser ;
fusionner ;
disparaître ;
se transformer.

Leur évolution dépend des décisions de leurs membres et des événements 
du monde.
⸻
IMPACT

Les communautés influencent :

les relations ;
les métiers ;
les traditions ;
l'économie ;
les conflits ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux communautés devra :

favoriser les interactions humaines ;
respecter leur évolution naturelle ;
créer des conséquences durables ;
renforcer l'identité locale ;
enrichir le monde vivant.
⸻
CRITÈRE DE VALIDATION

Cette mécanique permet-elle aux communautés d'évoluer naturellement et 
d'influencer le monde de manière crédible ?

Si la réponse est non, elle devra être repensée.
