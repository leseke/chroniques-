# GDB-030

> Les technologies, leurs infrastructures et leur évolution au service des habitants et des civilisations.

## Présentation

La série **GDB-030** décrit les systèmes technologiques de Chroniques. Elle définit les outils, les procédés, les réseaux et les infrastructures qui permettent aux civilisations d'exploiter les connaissances scientifiques afin de transformer leur environnement.

## Objectif

Répondre à la question :

> Comment les technologies sont-elles développées, utilisées et améliorées dans Chroniques ?

## Cette série couvre

- les fondements de la technologie ;
- les outils technologiques ;
- les procédés technologiques ;
- les infrastructures technologiques ;
- les innovations technologiques ;
- les réseaux technologiques ;
- les systèmes technologiques ;
- les normes technologiques ;
- l'évolution technologique ;
- la philosophie de la technologie.
