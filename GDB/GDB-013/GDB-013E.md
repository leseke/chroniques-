GDB-013E --- Les Matériaux de Construction

Version : 1.0
Statut : Officiel
Type : Construction & Habitat
⸻


OBJECTIF

Définir les principes fondamentaux des matériaux de construction dans 
Chroniques.

Les matériaux constituent la base de toute construction. Leurs 
propriétés influencent la solidité, la durabilité, l'esthétique, le coût 
et les possibilités architecturales.
⸻
PRINCIPE

Chaque matériau possède une identité propre.

Son utilisation dépend des ressources disponibles, des connaissances 
techniques, du climat, des traditions et des besoins des habitants.
⸻
LES GRANDES CATÉGORIES

Les matériaux comprennent notamment :

	⁃	le bois ;
	⁃	la pierre ;
	⁃	la terre ;
	⁃	les métaux ;
	⁃	le verre ;
	⁃	les fibres naturelles ;
	⁃	les matériaux composites.

Chaque catégorie présente des avantages et des contraintes.
⸻
ÉVOLUTION

Les matériaux évoluent grâce :

	⁃	aux innovations ;
	⁃	aux techniques de transformation ;
	⁃	aux échanges ;
	⁃	aux découvertes ;
	⁃	aux générations.

De nouveaux usages apparaissent sans rendre les anciens obsolètes.
⸻
IMPACT

Les matériaux influencent :

	⁃	les bâtiments ;
	⁃	les infrastructures ;
	⁃	les métiers ;
	⁃	l'économie ;
	⁃	les paysages ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux matériaux devra :

	1.	respecter la cohérence du monde ;
	2.	valoriser la diversité des ressources ;
	3.	produire des différences concrètes d'usage ;
	4.	interagir avec plusieurs systèmes ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Ce matériau enrichit-il réellement les possibilités de construction 
plutôt que de n'être qu'une variante esthétique ?

Si la réponse est non, elle devra être repensée.
⸻
