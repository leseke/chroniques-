GDB-013H --- Le Patrimoine

Version : 1.0
Statut : Officiel
Type : Construction & Habitat
⸻


OBJECTIF

Définir les principes fondamentaux du patrimoine dans Chroniques.

Le patrimoine représente l'ensemble des constructions, lieux et 
aménagements transmis au fil des générations. Il matérialise l'histoire 
du monde et les réalisations des habitants.
⸻
PRINCIPE

Le patrimoine est une mémoire vivante.

Chaque bâtiment, ouvrage ou lieu remarquable raconte une partie de 
l'histoire d'une famille, d'une communauté ou d'une civilisation.
⸻
LES FORMES DE PATRIMOINE

Le patrimoine peut comprendre :

	⁃	les habitations ;
	⁃	les monuments ;
	⁃	les infrastructures ;
	⁃	les quartiers historiques ;
	⁃	les ouvrages d'art ;
	⁃	les sites naturels aménagés ;
	⁃	les lieux symboliques.

Chaque élément possède une valeur propre.
⸻
ÉVOLUTION

Le patrimoine peut :

	⁃	être préservé ;
	⁃	être restauré ;
	⁃	changer d'usage ;
	⁃	être enrichi ;
	⁃	disparaître.

Son évolution dépend des choix des générations successives.
⸻
IMPACT

Le patrimoine influence :

	⁃	les communautés ;
	⁃	les cultures ;
	⁃	les échanges ;
	⁃	l'attractivité des territoires ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée au patrimoine devra :

	1.	valoriser la transmission ;
	2.	produire des conséquences durables ;
	3.	respecter l'histoire du monde ;
	4.	favoriser la restauration autant que la création ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle au patrimoine une véritable valeur 
historique et sociale plutôt qu'un simple rôle décoratif ?

Si la réponse est non, elle devra être repensée.
⸻
