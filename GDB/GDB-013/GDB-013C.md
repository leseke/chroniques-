GDB-013C --- Les Infrastructures

Version : 1.0
Statut : Officiel
Type : Construction & Habitat
⸻


OBJECTIF

Définir les principes fondamentaux des infrastructures dans Chroniques.

Les infrastructures relient les habitants, les bâtiments et les 
territoires. Elles permettent aux sociétés de se développer durablement 
en facilitant les déplacements, les échanges et l'organisation 
collective.
⸻
PRINCIPE

Une infrastructure est un aménagement conçu pour répondre à un besoin 
collectif.

Elle améliore le fonctionnement du monde tout en restant intégrée à son 
environnement.
⸻
LES GRANDES CATÉGORIES

Les infrastructures comprennent notamment :

	⁃	les routes ;
	⁃	les ponts ;
	⁃	les ports ;
	⁃	les réseaux d'eau ;
	⁃	les systèmes de transport ;
	⁃	les ouvrages de protection ;
	⁃	les espaces publics.

Chaque infrastructure possède une fonction précise.
⸻
ÉVOLUTION

Les infrastructures évoluent grâce :

	⁃	aux innovations ;
	⁃	aux ressources ;
	⁃	aux besoins des communautés ;
	⁃	aux générations ;
	⁃	aux transformations du territoire.

Elles peuvent être entretenues, agrandies ou remplacées.
⸻
IMPACT

Les infrastructures influencent :

	⁃	les échanges ;
	⁃	les métiers ;
	⁃	les déplacements ;
	⁃	les communautés ;
	⁃	l'économie ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux infrastructures devra :

	1.	répondre à un besoin crédible ;
	2.	évoluer avec le monde ;
	3.	interagir avec plusieurs systèmes ;
	4.	produire des conséquences durables ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette infrastructure améliore-t-elle réellement la vie du monde plutôt 
que d'être un simple élément décoratif ?

Si la réponse est non, elle devra être repensée.
⸻
