# GDB-013

> Les constructions et l'aménagement du monde.

## Présentation

La série **GDB-013** définit les principes liés aux habitats, aux bâtiments, aux infrastructures et à l'évolution des territoires.

## Objectif

Répondre à la question :

> Comment les habitants construisent-ils, développent-ils et entretiennent-ils leur environnement ?

## Cette série couvre

- les habitats ;
- les bâtiments ;
- les infrastructures ;
- l'aménagement des territoires ;
- les matériaux de construction ;
- l'architecture ;
- l'entretien ;
- le patrimoine ;
- l'évolution des territoires ;
- la philosophie de la construction.
