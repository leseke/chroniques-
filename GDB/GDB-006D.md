GDB-006D --- Les Découvertes

Version : 1.0 Statut : Officiel Type : Expérience Joueur
⸻
OBJECTIF

Définir les principes fondamentaux des découvertes dans Chroniques.

Les découvertes nourrissent la curiosité du joueur et donnent au monde 
le sentiment d'être plus vaste que ce qu'il connaît déjà.
⸻
PRINCIPE

Une découverte est une rencontre avec quelque chose d'inconnu.

Elle peut concerner un lieu, une personne, une ressource, un savoir, une 
culture, un phénomène ou une opportunité.
⸻
LA CURIOSITÉ COMME MOTEUR

Les découvertes doivent donner envie d'explorer davantage.

Le plaisir vient avant la récompense.

Le jeu privilégie l'émerveillement à l'accumulation.
⸻
DIVERSITÉ

Les découvertes peuvent être :

	⁃	géographiques ;
	⁃	historiques ;
	⁃	culturelles ;
	⁃	scientifiques ;
	⁃	humaines ;
	⁃	économiques ;
	⁃	naturelles.

Chaque catégorie enrichit une facette différente du monde.
⸻
ÉVOLUTION

Le même lieu peut révéler de nouvelles découvertes au fil des saisons, 
des générations ou des conséquences des actions du joueur.

Le monde conserve ainsi une capacité permanente à surprendre.
⸻
IMPACT

Les découvertes peuvent :

	⁃	ouvrir de nouvelles opportunités ;
	⁃	modifier un projet ;
	⁃	transformer une réputation ;
	⁃	enrichir les connaissances ;
	⁃	créer des histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux découvertes devra :

	1.	récompenser la curiosité ;
	2.	éviter les découvertes purement aléatoires ;
	3.	respecter la cohérence du monde ;
	4.	encourager l'exploration libre ;
	5.	enrichir durablement l'expérience du joueur.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle envie d'explorer pour découvrir plutôt 
que pour simplement obtenir une récompense ?

Si la réponse est non, elle devra être repensée.
⸻
