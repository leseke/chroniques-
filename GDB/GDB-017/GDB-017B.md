GDB-017B --- Les Croyances

Version : 1.0 Statut : Officiel Type : Religion & Croyances
⸻
OBJECTIF

Définir les principes fondamentaux des croyances dans Chroniques.

Les croyances représentent l'ensemble des convictions spirituelles, 
philosophiques ou culturelles qui influencent la manière dont les 
individus interprètent le monde, prennent des décisions et interagissent 
avec leur environnement.
⸻
PRINCIPE

Une croyance est une conviction partagée ou individuelle.

Elle peut être transmise par la famille, la communauté, une institution 
ou une expérience personnelle. Les croyances évoluent avec les 
connaissances, les événements et les générations.
⸻
LES FORMES DE CROYANCES

Les croyances peuvent notamment concerner :

	⁃	les divinités ;
	⁃	les forces naturelles ;
	⁃	les esprits ;
	⁃	l'origine du monde ;
	⁃	la vie après la mort ;
	⁃	les traditions ;
	⁃	les valeurs morales.

Chaque société développe son propre système de croyances.
⸻
ÉVOLUTION

Les croyances évoluent grâce :

	⁃	aux échanges culturels ;
	⁃	aux découvertes ;
	⁃	aux traditions ;
	⁃	aux conflits ;
	⁃	aux générations.

Elles peuvent se renforcer, se transformer ou disparaître.
⸻
IMPACT

Les croyances influencent :

	⁃	les comportements ;
	⁃	les communautés ;
	⁃	les religions ;
	⁃	les lois ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux croyances devra :

	1.	respecter la diversité des visions du monde ;
	2.	évoluer naturellement avec les sociétés ;
	3.	influencer les décisions sans les imposer ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle des croyances un moteur culturel vivant 
plutôt qu'un simple attribut de personnage ?

Si la réponse est non, elle devra être repensée.
