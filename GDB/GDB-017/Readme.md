# GDB-017

> Les religions, croyances et spiritualités présentes dans Chroniques.

## Présentation

La série **GDB-017** décrit les systèmes religieux, les croyances et les pratiques spirituelles qui influencent les habitants et les civilisations du monde.

## Objectif

Répondre à la question :

> Quel rôle jouent les croyances dans l'évolution des sociétés ?

## Cette série couvre

- les religions ;
- les croyances ;
- les divinités ;
- les rites ;
- les lieux sacrés ;
- les textes sacrés ;
- les représentants religieux ;
- les communautés religieuses ;
- les schismes ;
- la philosophie des religions.
