# GDB-006

> L'expérience émotionnelle recherchée pour le joueur.

---

# Présentation

La série **GDB-006** définit les besoins, les émotions et les sensations que Chroniques cherche à provoquer chez le joueur.

Contrairement aux séries qui décrivent les systèmes du jeu, celle-ci se concentre sur l'expérience vécue. Elle constitue une référence pour toutes les décisions de game design afin de garantir une expérience cohérente avec la vision du projet.

---

# Objectif

La série GDB-006 répond à la question :

> **Que doit ressentir le joueur tout au long de son aventure ?**

---

# Cette série couvre

- les besoins fondamentaux du joueur ;
- la motivation ;
- la progression perçue ;
- les découvertes ;
- l'exploration ;
- l'immersion ;
- les émotions ;
- les souvenirs marquants ;
- l'attachement au monde ;
- la satisfaction.

---

# Cette série ne couvre pas

Cette série ne décrit pas :

- les règles techniques ;
- les systèmes économiques ;
- les personnages ;
- les organisations ;
- l'architecture du moteur.

# Relations

Cette série influence directement l'ensemble des autres GDB. Chaque système doit contribuer à au moins un des objectifs d'expérience définis ici.

---

# Principes

- privilégier les émotions authentiques ;
- favoriser la curiosité ;
- éviter les récompenses artificielles ;
- renforcer l'attachement au monde ;
- maintenir une expérience cohérente avec la vision de Chroniques.
