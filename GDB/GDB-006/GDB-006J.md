GDB-006J --- La Satisfaction

Version : 1.0
Statut : Officiel
Type : Expérience Joueur
⸻


OBJECTIF

Définir les principes fondamentaux de la satisfaction du joueur dans 
Chroniques.

La satisfaction représente le sentiment durable d'avoir vécu une 
expérience qui avait du sens.
⸻
PRINCIPE

La satisfaction ne provient pas uniquement des récompenses obtenues.

Elle naît lorsque les efforts, les choix et les conséquences forment une 
histoire personnelle cohérente.
⸻
LES SOURCES DE SATISFACTION

La satisfaction peut provenir de :

	⁃	l'accomplissement d'un projet ;
	⁃	la maîtrise d'une compétence ;
	⁃	une découverte inattendue ;
	⁃	une relation construite avec le temps ;
	⁃	un héritage transmis ;
	⁃	un souvenir marquant.
⸻
UNE SATISFACTION DURABLE

Chroniques privilégie les réussites qui conservent leur valeur avec le 
temps.

Le jeu évite les gratifications instantanées destinées uniquement à 
provoquer un plaisir éphémère.
⸻
L'ÉCHEC A AUSSI UNE VALEUR

Un échec peut devenir une source de satisfaction lorsqu'il enseigne, 
ouvre une nouvelle voie ou enrichit l'histoire du joueur.

Le monde récompense l'expérience vécue, pas seulement la réussite.
⸻
IMPACT

Une satisfaction durable renforce :

	⁃	la motivation ;
	⁃	l'attachement au monde ;
	⁃	les souvenirs ;
	⁃	l'envie de transmettre son histoire ;
	⁃	le désir de commencer une nouvelle génération.
⸻
RÈGLES DE CONCEPTION

Toute mécanique devra :

	1.	privilégier le sens à la récompense immédiate ;
	2.	valoriser les efforts du joueur ;
	3.	accepter plusieurs formes de réussite ;
	4.	transformer les échecs en opportunités d'histoire ;
	5.	respecter les principes fondateurs de Chroniques.
⸻
CRITÈRE DE VALIDATION

Cette mécanique procure-t-elle une satisfaction qui restera mémorable 
bien après la fin de la session ?

Si la réponse est non, elle devra être repensée.
⸻
