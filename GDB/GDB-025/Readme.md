# GDB-025

> Les déplacements des habitants, des ressources et des informations.

---

# Présentation

La série **GDB-025** définit les différents moyens de transport présents dans Chroniques ainsi que leur influence sur le développement du monde.

---

# Objectif

La série GDB-025 répond à la question :

> **Comment les individus, les marchandises et les ressources circulent-ils ?**

---

# Cette série couvre

- les transports ;
- les véhicules ;
- les routes ;
- les voies maritimes ;
- les voies aériennes ;
- les réseaux ;
- les infrastructures ;
- la logistique ;
- la mobilité ;
- la philosophie des transports.
