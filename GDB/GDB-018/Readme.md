# GDB-018

> Le système judiciaire et l'application du droit.

## Présentation

La série **GDB-018** définit les règles qui encadrent la justice dans Chroniques. Elle décrit les institutions judiciaires, les procédures et les garanties offertes aux individus.

## Objectif

Répondre à la question :

> Comment le droit est-il appliqué dans Chroniques ?

## Cette série couvre

- la justice ;
- les lois ;
- les tribunaux ;
- les magistrats ;
- les procédures judiciaires ;
- les sanctions ;
- les droits des individus ;
- les recours ;
- les réformes du droit ;
- la philosophie de la justice.
