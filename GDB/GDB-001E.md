GDB-001E --- Le Temps

Version : 1.0
Statut : Officiel
Type : Principe Fondateur
⸻


OBJECTIF

Définir la philosophie du temps dans Chroniques.
⸻
LE TEMPS EST UNE RICHESSE

Le temps n'est pas un obstacle.

Il permet d'apprendre, transmettre, construire, découvrir et voir 
évoluer le monde.
⸻
LE TEMPS RESPECTE LE JOUEUR

Chroniques ne cherche jamais à créer une dépendance.

Le joueur ne doit pas ressentir l'obligation de se connecter pour ne pas 
"rater" le jeu.
⸻
LE REFUS DU FOMO

Aucune mécanique centrale ne doit reposer sur la peur de manquer une 
opportunité.

Les opportunités évoluent, mais le monde continue toujours d'en créer de 
nouvelles.
⸻
LE RYTHME APPARTIENT AU JOUEUR

Chaque joueur choisit librement son rythme.

Tous les rythmes sont légitimes.

Le jeu accompagne ce choix sans le récompenser ni le pénaliser.
⸻
LE TEMPS ENRICHIT LE MONDE

Les saisons changent.

Les générations se succèdent.

Les villes évoluent.

Les histoires progressent.

Le temps donne de la profondeur au monde.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée au temps devra :

	1.	respecter le temps réel du joueur ;
	2.	éviter les obligations artificielles ;
	3.	enrichir le monde plutôt que punir l'absence ;
	4.	favoriser la continuité des histoires.
⸻
CRITÈRE DE VALIDATION

Cette mécanique utilise-t-elle le temps pour enrichir l'expérience 
plutôt que pour contraindre le joueur ?

Si la réponse est non, elle devra être repensée.
