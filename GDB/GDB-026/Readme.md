# GDB-026

> Les systèmes permettant aux individus, aux organisations et aux civilisations d'échanger des informations.

## Présentation

La série **GDB-026** définit les mécanismes de communication dans Chroniques. Elle couvre les langages, les médias, les réseaux et les infrastructures permettant la circulation des informations entre les différents acteurs du monde.

## Objectif

Répondre à la question :

> Comment les informations circulent-elles entre les habitants et les civilisations ?

## Cette série couvre

- les fondements de la communication ;
- les langages ;
- les médias ;
- les réseaux de communication ;
- les systèmes de diffusion ;
- les protocoles de communication ;
- la sécurité des communications ;
- les flux d'information ;
- les centres de communication ;
- la philosophie de la communication.
