GDB-005B --- Les Ressources

Version : 1.0
Statut : Officiel
Type : Économie & Progression
⸻


OBJECTIF

Définir les principes fondamentaux des ressources dans Chroniques.

Les ressources constituent la matière première du monde vivant.
⸻
PRINCIPE

Toute ressource doit avoir une origine crédible, une utilité 
identifiable et une place logique dans son environnement.

Une ressource n'existe jamais uniquement pour être récoltée.
⸻
ORIGINE

Les ressources proviennent notamment :

	⁃	de la nature ;
	⁃	des animaux ;
	⁃	des cultures ;
	⁃	de la transformation ;
	⁃	du recyclage ;
	⁃	des conséquences du monde.
⸻
CYCLE DE VIE

Une ressource apparaît, est exploitée, transformée, consommée ou 
disparaît.

Certaines se renouvellent rapidement.

D'autres demandent du temps.
⸻
RARETÉ

La rareté dépend du monde.

Elle évolue naturellement selon les activités et les conséquences.
⸻
INTERDÉPENDANCE

Les ressources relient les métiers entre eux.

Chaque chaîne de production doit favoriser les échanges.
⸻
RÈGLES DE CONCEPTION

Toute ressource devra :

	1.	avoir une origine crédible ;
	2.	posséder plusieurs usages ;
	3.	interagir avec plusieurs métiers ;
	4.	évoluer avec le monde ;
	5.	enrichir les histoires émergentes.
⸻
CRITÈRE DE VALIDATION

Cette ressource enrichit-elle réellement l'écosystème du monde ?

Si la réponse est non, elle devra être repensée.
⸻
Fin du document

Statut : Validé -- Référence officielle.
