GDB-005A --- Les Métiers

Version : 1.0
Statut : Officiel
Type : Économie & Progression
⸻


OBJECTIF

Définir les principes fondamentaux qui gouvernent les métiers dans 
Chroniques.

Les métiers représentent une manière de vivre dans le monde avant d'être 
un moyen de gagner de l'argent.
⸻
PRINCIPE

Un métier est une activité durable permettant à un individu de créer de 
la valeur pour lui-même et pour le monde.

Chaque métier doit avoir une utilité crédible dans l'écosystème global.
⸻
LIBERTÉ

Le joueur choisit librement ses métiers.

Il peut :

	⁃	se spécialiser ;
	⁃	exercer plusieurs métiers ;
	⁃	changer de voie ;
	⁃	transmettre un savoir-faire.

Aucune profession n'est obligatoire.
⸻
INTERDÉPENDANCE

Les métiers ne doivent jamais fonctionner isolément.

Chaque métier produit, transforme ou utilise les ressources créées par 
d'autres.

Cette interdépendance alimente naturellement l'économie.
⸻
MAÎTRISE

La progression dans un métier repose sur :

	⁃	la pratique ;
	⁃	la qualité du travail ;
	⁃	l'expérience acquise ;
	⁃	la transmission.

L'excellence reste rare et valorisée.
⸻
ÉVOLUTION

Les métiers évoluent avec :

	⁃	les besoins du monde ;
	⁃	les innovations ;
	⁃	les générations ;
	⁃	les conséquences des actions.
⸻
RÈGLES DE CONCEPTION

Toute création de métier devra :

	1.	répondre à un besoin crédible ;
	2.	interagir avec plusieurs autres métiers ;
	3.	proposer plusieurs styles de pratique ;
	4.	créer des histoires émergentes ;
	5.	respecter les principes fondateurs de Chroniques.
⸻
CRITÈRE DE VALIDATION

Ce métier enrichit-il réellement le monde plutôt que d'ajouter une 
activité isolée ?

Si la réponse est non, il devra être repensé.
⸻
