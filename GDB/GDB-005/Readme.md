# GDB-005

> Les activités économiques et la création de valeur dans Chroniques.

---

# Présentation

La série **GDB-005** décrit le fonctionnement économique du monde de Chroniques.

Elle couvre les métiers, la production, les ressources, les échanges et les mécanismes qui permettent de créer, transformer et distribuer la valeur.

Cette série constitue la référence officielle pour l'économie du jeu.

---

# Objectif

La série GDB-005 répond à la question :

> **Comment les richesses sont-elles produites, échangées et valorisées dans Chroniques ?**

---

# Cette série couvre

- les métiers ;
- les ressources ;
- les chaînes de production ;
- les outils ;
- les produits ;
- les échanges ;
- les marchés ;
- la monnaie ;
- la valeur ;
- les investissements.

---

# Cette série ne couvre pas

Cette série ne décrit pas :

- les habitants ;
- les organisations politiques ;
- les mécaniques techniques ;
- l'architecture du moteur.

# Relations

Cette série dépend des principes de **GDB-001** et collabore étroitement avec les séries consacrées aux habitants, aux organisations et aux infrastructures.

---

# Principes

Les systèmes économiques doivent être :

- cohérents ;
- émergents ;
- équilibrés ;
- évolutifs ;
- interconnectés.
