GDB-014F --- Les Secrets

Version : 1.0
Statut : Officiel
Type : Exploration & Découverte
⸻


OBJECTIF

Définir les principes fondamentaux des secrets dans Chroniques.

Les secrets représentent les éléments cachés du monde qui ne se révèlent 
qu'aux explorateurs les plus curieux, observateurs ou persévérants. Ils 
renforcent le mystère et la profondeur de l'univers.
⸻
PRINCIPE

Un secret ne se trouve pas par hasard.

Il est découvert grâce à l'observation, aux connaissances, à 
l'exploration, aux interactions ou à des circonstances particulières.

Chaque secret possède une raison d'exister.
⸻
LES FORMES DE SECRETS

Les secrets peuvent notamment concerner :

	⁃	des lieux cachés ;
	⁃	des passages dissimulés ;
	⁃	des connaissances oubliées ;
	⁃	des objets uniques ;
	⁃	des événements rares ;
	⁃	des personnages mystérieux ;
	⁃	des phénomènes inexpliqués.

Chaque secret enrichit la compréhension du monde.
⸻
ÉVOLUTION

Les secrets évoluent grâce :

	⁃	aux découvertes ;
	⁃	aux expéditions ;
	⁃	aux recherches ;
	⁃	aux générations ;
	⁃	aux transformations du monde.

Certains peuvent disparaître tandis que d'autres émergent.
⸻
IMPACT

Les secrets influencent :

	⁃	les découvertes ;
	⁃	les histoires émergentes ;
	⁃	les innovations ;
	⁃	les civilisations ;
	⁃	l'exploration.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux secrets devra :

	1.	récompenser la curiosité ;
	2.	éviter les révélations arbitraires ;
	3.	conserver une part de mystère ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer le sentiment d'un monde riche et vivant.
⸻
CRITÈRE DE VALIDATION

Cette mécanique transforme-t-elle chaque secret en une découverte 
mémorable plutôt qu'en un simple objet caché ?

Si la réponse est non, elle devra être repensée.
⸻
