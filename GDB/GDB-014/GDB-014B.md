GDB-014B --- La Cartographie

Version : 1.0
Statut : Officiel
Type : Exploration & Découverte
⸻


OBJECTIF

Définir les principes fondamentaux de la cartographie dans Chroniques.

La cartographie permet de représenter, comprendre et transmettre la 
connaissance du monde. Elle accompagne l'exploration et conserve la 
mémoire des territoires découverts.
⸻
PRINCIPE

Une carte est une représentation évolutive du monde.

Elle s'enrichit grâce aux explorations, aux observations et aux échanges 
de connaissances. Aucune carte n'est parfaite ni définitivement achevée.
⸻
LES ÉLÉMENTS CARTOGRAPHIÉS

La cartographie peut représenter notamment :

	⁃	les territoires ;
	⁃	les biomes ;
	⁃	les reliefs ;
	⁃	les cours d'eau ;
	⁃	les routes ;
	⁃	les points d'intérêt ;
	⁃	les frontières.

Chaque information améliore la compréhension du monde.
⸻
ÉVOLUTION

La cartographie progresse grâce :

	⁃	aux explorations ;
	⁃	aux observations ;
	⁃	aux outils ;
	⁃	aux innovations ;
	⁃	aux générations.

Les cartes deviennent plus précises avec le temps.
⸻
IMPACT

La cartographie influence :

	⁃	les voyages ;
	⁃	les échanges ;
	⁃	les expéditions ;
	⁃	les découvertes ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée à la cartographie devra :

	1.	récompenser l'exploration ;
	2.	refléter fidèlement les découvertes ;
	3.	évoluer progressivement ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle de la carte un véritable outil d'exploration 
plutôt qu'un simple menu de navigation ?

Si la réponse est non, elle devra être repensée.
⸻
