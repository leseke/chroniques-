GDB-014G --- Les Ruines

Version : 1.0
Statut : Officiel
Type : Exploration & Découverte
⸻


OBJECTIF

Définir les principes fondamentaux des ruines dans Chroniques.

Les ruines témoignent des civilisations passées, des catastrophes, des 
conflits ou des époques oubliées. Elles constituent des lieux 
d'exploration riches en histoire, en mystères et en découvertes.
⸻
PRINCIPE

Une ruine est la trace d'un passé.

Elle raconte une histoire à travers son architecture, son état de 
conservation, les objets qu'elle renferme et les indices laissés par 
ceux qui l'ont occupée.

Chaque ruine possède une identité propre.
⸻
LES TYPES DE RUINES

Les ruines peuvent notamment être :

	⁃	d'anciennes cités ;
	⁃	des temples oubliés ;
	⁃	des forteresses abandonnées ;
	⁃	des villages disparus ;
	⁃	des laboratoires anciens ;
	⁃	des infrastructures délaissées ;
	⁃	des monuments effondrés.

Chaque type apporte un contexte unique.
⸻
ÉVOLUTION

Les ruines évoluent avec :

	⁃	le temps ;
	⁃	le climat ;
	⁃	la végétation ;
	⁃	les explorations ;
	⁃	les événements du monde.

Elles peuvent être redécouvertes, restaurées ou disparaître.
⸻
IMPACT

Les ruines influencent :

	⁃	les découvertes ;
	⁃	les recherches ;
	⁃	les civilisations ;
	⁃	les expéditions ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux ruines devra :

	1.	raconter une histoire crédible ;
	2.	récompenser l'observation ;
	3.	proposer des découvertes significatives ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer le sentiment d'ancienneté du monde.
⸻
CRITÈRE DE VALIDATION

Cette mécanique transforme-t-elle chaque ruine en un témoignage vivant 
du passé plutôt qu'en un simple donjon à explorer ?

Si la réponse est non, elle devra être repensée.
⸻
