# GDB-014

> L'exploration et la découverte du monde.

## Présentation

La série **GDB-014** décrit tous les mécanismes liés à l'exploration de Chroniques, depuis les déplacements jusqu'à la découverte de lieux rares et de secrets.

## Objectif

Répondre à la question :

> Comment encourager une exploration libre, naturelle et gratifiante ?

## Cette série couvre

- l'exploration ;
- la cartographie ;
- les voyages ;
- les expéditions ;
- les découvertes ;
- les secrets ;
- les ruines ;
- les points d'intérêt ;
- les phénomènes rares ;
- la philosophie de l'exploration.
