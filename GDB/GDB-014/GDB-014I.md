GDB-014I --- Les Phénomènes Rares

Version : 1.0
Statut : Officiel
Type : Exploration & Découverte
⸻


OBJECTIF

Définir les principes fondamentaux des phénomènes rares dans Chroniques.

Les phénomènes rares représentent des événements, manifestations ou 
conditions exceptionnelles qui enrichissent le monde et récompensent les 
explorateurs attentifs. Leur apparition renforce le sentiment d'évoluer 
dans un univers vivant et imprévisible.
⸻
PRINCIPE

Un phénomène rare est un événement exceptionnel.

Il apparaît selon des conditions naturelles, historiques ou 
environnementales cohérentes. Sa rareté lui confère une valeur 
particulière sans en faire une simple récompense aléatoire.
⸻
LES TYPES DE PHÉNOMÈNES RARES

Les phénomènes rares peuvent notamment être :

	⁃	des événements climatiques exceptionnels ;
	⁃	des migrations inhabituelles d'espèces ;
	⁃	des floraisons extraordinaires ;
	⁃	des alignements célestes ;
	⁃	des manifestations géologiques ;
	⁃	des anomalies naturelles ;
	⁃	des événements culturels uniques.

Chaque phénomène possède son propre contexte.
⸻
ÉVOLUTION

Les phénomènes rares évoluent grâce :

	⁃	au temps ;
	⁃	aux saisons ;
	⁃	aux cycles naturels ;
	⁃	aux transformations du monde ;
	⁃	aux générations.

Leur apparition demeure exceptionnelle mais cohérente.
⸻
IMPACT

Les phénomènes rares influencent :

	⁃	les découvertes ;
	⁃	les expéditions ;
	⁃	les recherches ;
	⁃	les communautés ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux phénomènes rares devra :

	1.	préserver leur caractère exceptionnel ;
	2.	respecter la cohérence du monde ;
	3.	récompenser l'observation et la curiosité ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer le sentiment d'un monde vivant.
⸻
CRITÈRE DE VALIDATION

Cette mécanique transforme-t-elle chaque phénomène rare en un 
événement mémorable plutôt qu'en une apparition aléatoire sans 
signification ?

Si la réponse est non, elle devra être repensée.
