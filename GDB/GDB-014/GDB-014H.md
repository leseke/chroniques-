GDB-014H --- Les Points d'Intérêt

Version : 1.0
Statut : Officiel
Type : Exploration & Découverte
⸻


OBJECTIF

Définir les principes fondamentaux des points d'intérêt dans Chroniques.

Les points d'intérêt sont des lieux remarquables qui attirent 
naturellement l'attention des explorateurs. Ils donnent au monde une 
identité forte et encouragent la découverte sans imposer un parcours 
unique.
⸻
PRINCIPE

Un point d'intérêt possède une valeur particulière.

Il peut être remarquable par son histoire, son architecture, sa 
géographie, sa biodiversité, sa culture ou les événements qui s'y 
déroulent.

Chaque point d'intérêt raconte quelque chose du monde.
⸻
LES TYPES DE POINTS D'INTÉRÊT

Les points d'intérêt peuvent notamment être :

	⁃	des monuments ;
	⁃	des paysages exceptionnels ;
	⁃	des villages isolés ;
	⁃	des sites naturels ;
	⁃	des lieux historiques ;
	⁃	des constructions uniques ;
	⁃	des phénomènes rares.

Chaque lieu possède une identité propre.
⸻
ÉVOLUTION

Les points d'intérêt évoluent grâce :

	⁃	au temps ;
	⁃	aux habitants ;
	⁃	aux événements ;
	⁃	aux catastrophes naturelles ;
	⁃	aux générations.

Ils peuvent apparaître, se transformer ou disparaître.
⸻
IMPACT

Les points d'intérêt influencent :

	⁃	les voyages ;
	⁃	les découvertes ;
	⁃	les communautés ;
	⁃	les échanges ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux points d'intérêt devra :

	1.	encourager l'exploration libre ;
	2.	raconter une histoire crédible ;
	3.	proposer une identité unique ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique transforme-t-elle chaque point d'intérêt en un lieu 
mémorable plutôt qu'en un simple marqueur sur une carte ?

Si la réponse est non, elle devra être repensée.
