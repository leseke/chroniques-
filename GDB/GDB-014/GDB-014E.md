GDB-014E --- Les Découvertes

Version : 1.0
Statut : Officiel
Type : Exploration & Découverte
⸻


OBJECTIF

Définir les principes fondamentaux des découvertes dans Chroniques.

Les découvertes représentent les connaissances, lieux, ressources ou 
phénomènes révélés par l'exploration. Elles enrichissent durablement la 
compréhension du monde et ouvrent de nouvelles possibilités.
⸻
PRINCIPE

Une découverte est une révélation du monde.

Elle peut être individuelle ou collective et conserve une valeur au-delà 
du moment où elle est réalisée. Chaque découverte contribue à l'histoire 
vivante de Chroniques.
⸻
LES FORMES DE DÉCOUVERTE

Les découvertes peuvent notamment concerner :

	⁃	des territoires inconnus ;
	⁃	des espèces vivantes ;
	⁃	des ressources rares ;
	⁃	des civilisations ;
	⁃	des technologies ;
	⁃	des phénomènes naturels ;
	⁃	des savoirs oubliés.

Chaque découverte possède une utilité ou une signification.
⸻
ÉVOLUTION

Les découvertes évoluent grâce :

	⁃	aux explorations ;
	⁃	aux recherches ;
	⁃	aux expéditions ;
	⁃	aux échanges de connaissances ;
	⁃	aux générations.

Le monde dévoile progressivement sa richesse.
⸻
IMPACT

Les découvertes influencent :

	⁃	les innovations ;
	⁃	les métiers ;
	⁃	les échanges ;
	⁃	les civilisations ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux découvertes devra :

	1.	récompenser la curiosité ;
	2.	produire des conséquences durables ;
	3.	enrichir les connaissances du monde ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer le sentiment d'émerveillement.
⸻
CRITÈRE DE VALIDATION

Cette mécanique transforme-t-elle chaque découverte en un événement 
marquant plutôt qu'en une simple récompense ?

Si la réponse est non, elle devra être repensée.
⸻
