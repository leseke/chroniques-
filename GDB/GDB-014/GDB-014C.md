GDB-014C --- Les Voyages

Version : 1.0
Statut : Officiel
Type : Exploration & Découverte
⸻


OBJECTIF

Définir les principes fondamentaux des voyages dans Chroniques.

Les voyages permettent aux personnages de parcourir le monde, de relier 
les territoires, de découvrir de nouvelles cultures et de favoriser les 
échanges. Ils constituent un pilier de l'exploration.
⸻
PRINCIPE

Voyager consiste à quitter un lieu connu pour en découvrir un autre.

Chaque déplacement représente une expérience pouvant apporter des 
connaissances, des rencontres, des opportunités ou des défis.
⸻
LES FORMES DE VOYAGE

Les voyages peuvent notamment avoir pour objectif :

	⁃	l'exploration ;
	⁃	le commerce ;
	⁃	la recherche ;
	⁃	la diplomatie ;
	⁃	le pèlerinage ;
	⁃	la migration ;
	⁃	l'aventure.

Chaque voyage possède une raison d'être.
⸻
ÉVOLUTION

Les voyages évoluent grâce :

	⁃	aux moyens de transport ;
	⁃	aux cartes ;
	⁃	aux infrastructures ;
	⁃	aux connaissances ;
	⁃	aux innovations.

Le monde devient progressivement plus accessible sans perdre son 
mystère.
⸻
IMPACT

Les voyages influencent :

	⁃	les échanges ;
	⁃	les cultures ;
	⁃	les découvertes ;
	⁃	les métiers ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux voyages devra :

	1.	valoriser la découverte ;
	2.	rendre les déplacements significatifs ;
	3.	produire des conséquences crédibles ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle du voyage une véritable aventure plutôt 
qu'un simple déplacement d'un point A à un point B ?

Si la réponse est non, elle devra être repensée.
⸻
