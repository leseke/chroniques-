GDB-014A --- L'Exploration

Version : 1.0
Statut : Officiel
Type : Exploration & Découverte
⸻


OBJECTIF

Définir les principes fondamentaux de l'exploration dans Chroniques.

L'exploration permet aux personnages de découvrir le monde, d'élargir 
leurs connaissances et de créer de nouvelles opportunités. Elle 
constitue l'un des principaux moteurs de l'aventure.
⸻
PRINCIPE

Explorer consiste à aller au-delà du connu.

Chaque voyage est l'occasion de découvrir de nouveaux lieux, ressources, 
cultures, phénomènes ou savoirs. L'exploration récompense la curiosité 
plutôt que la simple progression.
⸻
LES FORMES D'EXPLORATION

L'exploration peut notamment concerner :

	⁃	les territoires ;
	⁃	les biomes ;
	⁃	les civilisations ;
	⁃	les ressources ;
	⁃	les phénomènes naturels ;
	⁃	les lieux oubliés ;
	⁃	les connaissances.

Chaque découverte enrichit la compréhension du monde.
⸻
ÉVOLUTION

L'exploration évolue grâce :

	⁃	aux compétences ;
	⁃	aux outils ;
	⁃	aux cartes ;
	⁃	aux moyens de transport ;
	⁃	aux connaissances accumulées.

Le monde révèle progressivement sa profondeur.
⸻
IMPACT

L'exploration influence :

	⁃	les métiers ;
	⁃	les échanges ;
	⁃	les innovations ;
	⁃	les histoires émergentes ;
	⁃	les générations.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée à l'exploration devra :

	1.	récompenser la curiosité ;
	2.	éviter les parcours entièrement linéaires ;
	3.	produire des découvertes significatives ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer le sentiment d'un monde vivant.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle envie d'explorer le monde pour le plaisir 
de découvrir plutôt que pour suivre uniquement un objectif ?

Si la réponse est non, elle devra être repensée.
⸻
