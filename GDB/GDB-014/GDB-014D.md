GDB-014D --- Les Expéditions

Version : 1.0
Statut : Officiel
Type : Exploration & Découverte
⸻


OBJECTIF

Définir les principes fondamentaux des expéditions dans Chroniques.

Les expéditions représentent des explorations organisées ayant un 
objectif précis. Elles mobilisent des ressources, des compétences et une 
préparation adaptée afin de repousser les limites du monde connu.
⸻
PRINCIPE

Une expédition est une entreprise planifiée.

Elle vise à découvrir, étudier, cartographier, récupérer ou atteindre un 
objectif situé au-delà des territoires familiers.

Chaque expédition comporte des opportunités comme des risques.
⸻
LES TYPES D'EXPÉDITION

Les expéditions peuvent notamment concerner :

	⁃	l'exploration de territoires inconnus ;
	⁃	la recherche scientifique ;
	⁃	l'étude des écosystèmes ;
	⁃	la découverte de civilisations ;
	⁃	la récupération de ressources rares ;
	⁃	les missions archéologiques ;
	⁃	les phénomènes exceptionnels.

Chaque objectif nécessite une préparation adaptée.
⸻
ÉVOLUTION

Les expéditions évoluent grâce :

	⁃	aux connaissances ;
	⁃	aux cartes ;
	⁃	aux moyens de transport ;
	⁃	aux innovations ;
	⁃	à l'expérience des explorateurs.

Chaque réussite enrichit les suivantes.
⸻
IMPACT

Les expéditions influencent :

	⁃	les découvertes ;
	⁃	les échanges ;
	⁃	les innovations ;
	⁃	les métiers ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux expéditions devra :

	1.	récompenser la préparation ;
	2.	proposer des risques crédibles ;
	3.	offrir plusieurs approches ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer le sentiment d'aventure.
⸻
CRITÈRE DE VALIDATION

Cette mécanique transforme-t-elle chaque expédition en une aventure 
mémorable plutôt qu'en une simple mission scénarisée ?

Si la réponse est non, elle devra être repensée.
