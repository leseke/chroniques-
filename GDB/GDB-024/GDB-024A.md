GDB-024A --- Les Sources d'Énergie

Version : 1.0
Statut : Officiel
Type : Énergie
⸻


OBJECTIF

Définir les principes fondamentaux des sources d'énergie dans 
Chroniques.

Les sources d'énergie fournissent la puissance nécessaire au 
fonctionnement des êtres vivants, des outils, des machines, des 
infrastructures et des technologies. Elles constituent la base de tout 
développement économique, scientifique et industriel.
⸻
PRINCIPE

Une source d'énergie produit ou libère une énergie exploitable.

Chaque source possède des caractéristiques propres : disponibilité, 
rendement, stabilité, coût, impact environnemental et niveau 
technologique requis. Les civilisations peuvent utiliser une ou 
plusieurs sources simultanément.
⸻
LES PRINCIPALES SOURCES

Les sources d'énergie peuvent notamment être :

musculaire ;
animale ;
hydraulique ;
éolienne ;
solaire ;
géothermique ;
chimique ;
nucléaire ;
magique (selon les règles du monde).

Leur utilisation dépend du niveau de développement de la civilisation.
⸻
ÉVOLUTION

Les sources d'énergie évoluent grâce :

aux découvertes scientifiques ;
aux innovations technologiques ;
aux besoins des sociétés ;
aux ressources disponibles ;
aux générations.

De nouvelles sources peuvent apparaître tandis que d'autres deviennent 
obsolètes.
⸻
IMPACT

Les sources d'énergie influencent :

l'économie ;
l'industrie ;
les transports ;
les infrastructures ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux sources d'énergie devra :

refléter leurs avantages et limites ;
encourager la diversification énergétique ;
produire des conséquences crédibles ;
interagir avec les autres systèmes ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle des sources d'énergie un fondement du 
développement des civilisations plutôt qu'une simple ressource 
consommable ?

Si la réponse est non, elle devra être repensée.
