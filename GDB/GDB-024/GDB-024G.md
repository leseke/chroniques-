GDB-024G --- La Consommation d'Énergie

Version : 1.0\
Statut : Officiel\
Type : Énergie

------------------------------------------------------------------------

# OBJECTIF

Définir les principes fondamentaux de la consommation d'énergie dans
Chroniques.

La consommation d'énergie représente l'utilisation des ressources
énergétiques par les individus, les machines, les bâtiments, les
infrastructures et les civilisations. Elle conditionne le fonctionnement
quotidien de l'ensemble du monde.

------------------------------------------------------------------------

# PRINCIPE

Toute activité nécessite une quantité d'énergie adaptée à son
fonctionnement.

Chaque consommateur possède des besoins énergétiques propres qui varient
selon son niveau technologique, son activité, son efficacité et son
environnement. Une gestion maîtrisée de la consommation améliore la
résilience et la prospérité.

------------------------------------------------------------------------

# LES PRINCIPAUX CONSOMMATEURS

L'énergie peut être consommée par :

-   les habitations ;
-   les infrastructures publiques ;
-   les industries ;
-   les transports ;
-   les équipements scientifiques ;
-   les systèmes militaires ;
-   les technologies propres à Chroniques.

Chaque catégorie présente des profils de consommation différents.

------------------------------------------------------------------------

# ÉVOLUTION

La consommation évolue grâce :

-   aux innovations technologiques ;
-   à l'amélioration des rendements ;
-   aux changements démographiques ;
-   aux politiques énergétiques ;
-   aux générations.

Les besoins peuvent augmenter, diminuer ou être optimisés.

------------------------------------------------------------------------

# IMPACT

La consommation d'énergie influence :

-   l'économie ;
-   la production ;
-   les réseaux énergétiques ;
-   l'environnement ;
-   les histoires émergentes.

------------------------------------------------------------------------

# RÈGLES DE CONCEPTION

Toute mécanique liée à la consommation d'énergie devra :

1.  refléter les besoins réels des différents utilisateurs ;
2.  encourager l'efficacité énergétique ;
3.  produire des conséquences crédibles en cas de pénurie ou de
    surconsommation ;
4.  interagir avec tous les systèmes énergétiques ;
5.  renforcer l'immersion.

------------------------------------------------------------------------
