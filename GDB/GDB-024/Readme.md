# GDB-024

> Les sources d'énergie et leur utilisation.

---

# Présentation

La série **GDB-024** décrit les ressources énergétiques, leur production, leur stockage et leur consommation dans Chroniques.

---

# Objectif

La série GDB-024 répond à la question :

> **Comment l'énergie est-elle produite, distribuée et utilisée ?**

---

# Cette série couvre

- les sources d'énergie ;
- la production ;
- le stockage ;
- la distribution ;
- la consommation ;
- le rendement ;
- les infrastructures énergétiques ;
- les innovations énergétiques ;
- les impacts ;
- la philosophie de l'énergie.
