GDB-024B --- Les Ressources Énergétiques

Version : 1.0
Statut : Officiel
Type : Énergie
⸻


OBJECTIF

Définir les principes fondamentaux des ressources énergétiques dans 
Chroniques.

Les ressources énergétiques sont les matières ou phénomènes exploitables 
permettant de produire de l'énergie. Elles déterminent le potentiel 
énergétique d'une région, d'une civilisation ou d'une infrastructure et 
conditionnent son développement à long terme.
⸻
PRINCIPE

Une ressource énergétique est une source exploitable permettant de 
générer de l'énergie.

Chaque ressource possède une disponibilité, une qualité, une facilité 
d'extraction, un rendement, un coût d'exploitation et un impact 
environnemental. Certaines sont renouvelables, d'autres limitées ou 
épuisables.
⸻
LES PRINCIPALES RESSOURCES

Les ressources énergétiques peuvent notamment être :

le bois ;
le charbon ;
le pétrole ;
le gaz naturel ;
l'uranium ;
l'eau ;
le vent ;
le rayonnement solaire ;
la chaleur géothermique ;
toute ressource spécifique au monde de Chroniques.

Leur répartition influence directement l'équilibre des civilisations.
⸻
ÉVOLUTION

Les ressources énergétiques évoluent grâce :

aux découvertes de nouveaux gisements ;
aux progrès des techniques d'extraction ;
aux innovations technologiques ;
aux politiques de gestion ;
aux besoins des sociétés.

Certaines ressources peuvent devenir rares ou être remplacées par 
d'autres.
⸻
IMPACT

Les ressources énergétiques influencent :

l'économie ;
l'industrie ;
les transports ;
les infrastructures ;
les conflits et les coopérations entre civilisations.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux ressources énergétiques devra :

représenter leur abondance ou leur rareté ;
encourager une gestion stratégique ;
produire des conséquences crédibles sur le développement ;
interagir avec les autres systèmes ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle des ressources énergétiques un enjeu 
stratégique durable plutôt qu'un simple stock de matières premières ?

Si la réponse est non, elle devra être repensée.
