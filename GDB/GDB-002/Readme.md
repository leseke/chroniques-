# GDB-002

> Les dynamiques de vie qui façonnent l'expérience du joueur.

---

# Présentation

La série **GDB-002** décrit la manière dont le monde évolue autour du joueur au fil du temps.

Elle définit les mécanismes qui rendent chaque partie unique grâce aux événements, aux opportunités, aux projets, aux conséquences des actions et aux histoires émergentes.

Cette série constitue le cœur de la simulation vivante de Chroniques.

---

# Objectif

La série GDB-002 répond à la question :

> **Comment le monde produit-il naturellement une histoire unique pour chaque partie ?**

Elle décrit les principes qui permettent à la simulation de générer une expérience cohérente, crédible et émergente.

---

# Cette série couvre

La série documente notamment :

- le monde personnel du joueur ;
- la mémoire du monde ;
- les conséquences des actions ;
- les événements dynamiques ;
- les opportunités rencontrées ;
- les projets de vie ;
- les instants marquants ;
- les histoires émergentes ;
- le rythme de progression ;
- les choix significatifs.

---

# Cette série ne couvre pas

Les éléments suivants sont documentés dans d'autres séries :

- les fondations philosophiques du projet ;
- les systèmes économiques ;
- les métiers ;
- les personnages ;
- l'implémentation technique ;
- l'architecture du moteur.

# Relations

Cette série s'appuie sur les concepts définis dans **GDB-001**.

Elle interagit avec les séries consacrées aux personnages, aux systèmes, à l'économie et à l'environnement afin de produire une simulation cohérente.

---

# Principes

Les documents de cette série doivent :

- favoriser l'émergence plutôt que le script ;
- garantir la cohérence des conséquences ;
- préserver la liberté du joueur ;
- éviter les événements artificiels ;
- renforcer la sensation d'un monde vivant.

---

# Évolution

Cette série évoluera avec l'enrichissement des systèmes de simulation.

Toute nouvelle mécanique devra renforcer la capacité du monde à produire des histoires émergentes sans remettre en cause les principes fondateurs de Chroniques.
