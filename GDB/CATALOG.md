# GDB CATALOG

> Catalogue officiel de la Game Design Bible de Chroniques.

---

# Présentation

Le présent catalogue constitue le point d'entrée officiel de la bibliothèque GDB.

Chaque série correspond à un domaine unique de connaissance. Avant de créer un nouveau document, ce catalogue doit être consulté afin d'éviter les duplications et de garantir une organisation cohérente de la documentation.

# Recherche d'information

Avant toute création de document :

1. Identifier le sujet.
2. Localiser la série correspondante.
3. Vérifier si un document existe déjà.
4. Compléter le document existant si nécessaire.
5. Créer un nouveau document uniquement si aucun ne couvre le sujet.

---

# Règles

- Une seule série est responsable d'un domaine.
- Une information ne doit pas être dupliquée.
- Les liens entre séries remplacent la copie de contenu.
- Les README de chaque série constituent la référence documentaire.
