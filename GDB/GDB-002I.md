GDB-002I --- Le Rythme du Joueur

Version : 1.0 Statut : Officiel Type : Fondations du 
Gameplay
⸻
OBJECTIF

Définir les principes qui gouvernent le rythme de jeu dans Chroniques.

Le rythme appartient au joueur.

Le rôle du jeu est de l'accompagner, jamais de l'imposer.
⸻
LE RYTHME EST UN CHOIX

Chaque joueur est libre d'adopter son propre rythme.

Un joueur contemplatif, un bâtisseur, un commerçant ou un explorateur 
doivent tous pouvoir vivre une aventure complète.

Aucun rythme n'est considéré comme supérieur.
⸻
AUCUNE COURSE PERMANENTE

Chroniques refuse les mécaniques qui obligent le joueur à accélérer 
constamment.

La progression ne doit jamais dépendre d'une pression artificielle.
⸻
LE MONDE S'ADAPTE

Le monde continue d'évoluer.

Cependant, il produit continuellement de nouvelles possibilités.

Le joueur qui revient après une pause retrouve un monde vivant, pas un 
monde qui l'a définitivement dépassé.
⸻
LE RESPECT DU TEMPS

Le temps investi possède une valeur.

Le temps non investi ne doit jamais devenir une punition.

Le jeu récompense la présence par l'expérience vécue, jamais par la peur 
de manquer.
⸻
LES DIFFÉRENTS RYTHMES

Chroniques reconnaît notamment :

	⁃	le rythme contemplatif ;
	⁃	le rythme équilibré ;
	⁃	le rythme ambitieux.

Tous doivent rester viables.
⸻
RÈGLES DE CONCEPTION

Toute mécanique devra :

	1.	respecter le rythme choisi par le joueur ;
	2.	éviter les obligations quotidiennes ;
	3.	limiter le FOMO ;
	4.	permettre une reprise naturelle après une absence ;
	5.	préserver la liberté d'organisation.
⸻
CRITÈRE DE VALIDATION

Cette mécanique respecte-t-elle le rythme du joueur sans créer de 
pression artificielle ?

Si la réponse est non, elle devra être repensée.
