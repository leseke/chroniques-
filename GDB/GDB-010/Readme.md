# GDB-010

> Les écosystèmes et le monde naturel.

---

# Présentation

La série **GDB-010** définit le fonctionnement du monde vivant de Chroniques.

Elle décrit les écosystèmes, les espèces, le climat, les ressources naturelles et les équilibres qui régissent la nature.

---

# Objectif

La série GDB-010 répond à la question :

> **Comment fonctionne le monde naturel de Chroniques ?**

---

# Cette série couvre

- le monde vivant ;
- les écosystèmes ;
- la faune ;
- la flore ;
- le climat ;
- les biomes ;
- les catastrophes naturelles ;
- les ressources naturelles ;
- les équilibres naturels ;
- la philosophie du monde vivant.

---

# Cette série ne couvre pas

- les habitants ;
- les entreprises ;
- l'économie ;
- les systèmes techniques.

# Relations

Cette série fournit les bases environnementales utilisées par les habitants, les sociétés et les systèmes économiques.

---

# Principes

- nature autonome ;
- équilibres dynamiques ;
- biodiversité cohérente ;
- interactions permanentes entre les espèces et leur environnement.
