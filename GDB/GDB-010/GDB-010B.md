GDB-010B --- Les Écosystèmes

Version : 1.0
Statut : Officiel
Type : Monde Vivant
⸻


OBJECTIF

Définir les principes fondamentaux des écosystèmes dans Chroniques.

Les écosystèmes représentent les interactions entre les êtres vivants, 
les ressources et leur environnement. Ils assurent l'équilibre naturel 
du monde.
⸻
PRINCIPE

Un écosystème est un ensemble vivant.

Chaque élément influence les autres : une modification locale peut 
produire des conséquences à plus grande échelle.

Le monde recherche un équilibre sans jamais être figé.
⸻
COMPOSANTES

Un écosystème comprend notamment :

la faune ;
la flore ;
les ressources ;
le climat ;
les sols ;
les cours d'eau ;
les habitants.

Tous ces éléments interagissent en permanence.
⸻
ÉVOLUTION

Les écosystèmes évoluent avec :

les saisons ;
le climat ;
les catastrophes naturelles ;
les activités humaines ;
les espèces vivantes.

Ils peuvent se régénérer, s'appauvrir ou se transformer.
⸻
IMPACT

Les écosystèmes influencent :

les métiers ;
les ressources ;
les déplacements ;
les civilisations ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux écosystèmes devra :

respecter les équilibres naturels ;
produire des conséquences crédibles ;
évoluer avec le temps ;
favoriser les interactions entre systèmes ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle le sentiment d'un écosystème vivant et 
interdépendant plutôt que d'un simple décor naturel ?

Si la réponse est non, elle devra être repensée.
