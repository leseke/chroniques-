GDB-010I --- Les Équilibres Naturels

Version : 1.0
Statut : Officiel
Type : Monde Vivant
⸻


OBJECTIF

Définir les principes fondamentaux des équilibres naturels dans 
Chroniques.

Les équilibres naturels assurent la stabilité et l'évolution du monde. 
Ils résultent des interactions permanentes entre les êtres vivants, les 
ressources, le climat et les activités des civilisations.
⸻
PRINCIPE

Aucun élément du monde n'existe isolément.

Toute modification importante produit des conséquences directes ou 
indirectes sur d'autres systèmes.

L'équilibre est dynamique, jamais immobile.
⸻
LES FACTEURS D'ÉQUILIBRE

Les équilibres naturels dépendent notamment de :

la biodiversité ;
la disponibilité des ressources ;
le climat ;
les saisons ;
les interactions entre espèces ;
les activités humaines.
⸻
ÉVOLUTION

Les équilibres évoluent continuellement.

Ils peuvent être renforcés, fragilisés ou transformés selon les 
événements et les décisions prises dans le monde.

Les écosystèmes tendent naturellement vers un nouvel équilibre.
⸻
IMPACT

Les équilibres naturels influencent :

les écosystèmes ;
les ressources ;
les biomes ;
les civilisations ;
les métiers ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux équilibres naturels devra :

respecter les interactions écologiques ;
produire des conséquences crédibles ;
évoluer progressivement ;
éviter les changements arbitraires ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle le sentiment d'un monde qui s'autorégule 
naturellement plutôt que d'un système artificiel ?

Si la réponse est non, elle devra être repensée.
