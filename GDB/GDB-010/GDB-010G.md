GDB-010G --- Les Catastrophes Naturelles

Version : 1.0
Statut : Officiel
Type : Monde Vivant
⸻


OBJECTIF

Définir les principes fondamentaux des catastrophes naturelles dans 
Chroniques.

Les catastrophes naturelles représentent les grands phénomènes de la 
nature qui transforment le monde. Elles participent à son évolution sans 
être conçues comme de simples obstacles pour le joueur.
⸻
PRINCIPE

Une catastrophe naturelle est un événement exceptionnel issu du 
fonctionnement normal du monde.

Elle modifie durablement les paysages, les ressources, les sociétés et 
les écosystèmes.
⸻
TYPES DE CATASTROPHES

Les catastrophes peuvent notamment prendre la forme de :

tempêtes ;
inondations ;
sécheresses ;
incendies naturels ;
glissements de terrain ;
éruptions volcaniques ;
séismes.

Chaque phénomène possède ses propres causes et conséquences.
⸻
ÉVOLUTION

Les catastrophes peuvent :

apparaître naturellement ;
modifier un territoire ;
créer de nouvelles opportunités ;
faire évoluer les écosystèmes ;
influencer les générations futures.

Le monde se reconstruit progressivement après leur passage.
⸻
IMPACT

Les catastrophes influencent :

les biomes ;
les ressources ;
les communautés ;
les métiers ;
les routes ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux catastrophes naturelles devra :

respecter la cohérence du monde ;
produire des conséquences durables ;
offrir autant d'opportunités que de difficultés ;
éviter les événements arbitraires ;
renforcer le caractère vivant du monde.
⸻
CRITÈRE DE VALIDATION

Cette mécanique transforme-t-elle les catastrophes naturelles en 
moteurs d'évolution du monde plutôt qu'en simples pénalités ?

Si la réponse est non, elle devra être repensée.
