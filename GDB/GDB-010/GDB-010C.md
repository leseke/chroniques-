GDB-010C --- La Faune

Version : 1.0
Statut : Officiel
Type : Monde Vivant
⸻


OBJECTIF

Définir les principes fondamentaux de la faune dans Chroniques.

La faune constitue une composante essentielle du monde vivant. Les 
animaux participent aux équilibres naturels et possèdent leurs propres 
comportements, besoins et cycles de vie.
⸻
PRINCIPE

Les animaux existent pour eux-mêmes avant d'exister pour le joueur.

Ils naissent, grandissent, se déplacent, se reproduisent et 
interagissent avec leur environnement selon leur espèce.
⸻
COMPORTEMENTS

La faune peut notamment :

se nourrir ;
migrer ;
former des groupes ;
défendre un territoire ;
fuir un danger ;
coopérer ;
entrer en compétition.

Chaque espèce possède des comportements cohérents.
⸻
ÉVOLUTION

Les populations animales évoluent selon :

les saisons ;
les ressources disponibles ;
les prédateurs ;
les activités humaines ;
les changements climatiques.
⸻
IMPACT

La faune influence :

les écosystèmes ;
les ressources ;
les métiers ;
l'exploration ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée à la faune devra :

respecter la cohérence biologique ;
favoriser des comportements crédibles ;
évoluer avec le monde ;
produire des conséquences naturelles ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle le sentiment d'une faune vivante plutôt 
que d'animaux servant uniquement de ressources ?

Si la réponse est non, elle devra être repensée.
⸻
