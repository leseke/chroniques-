GDB-010E --- Le Climat

Version : 1.0
Statut : Officiel
Type : Monde Vivant
⸻


OBJECTIF

Définir les principes fondamentaux du climat dans Chroniques.

Le climat façonne durablement les paysages, les écosystèmes, les 
civilisations et les modes de vie. Il constitue l'un des moteurs 
naturels de l'évolution du monde.
⸻
PRINCIPE

Le climat est un système vivant.

Il évolue selon des cycles naturels et influence continuellement les 
êtres vivants ainsi que leurs activités.

Il ne sert jamais uniquement de décor.
⸻
LES COMPOSANTES DU CLIMAT

Le climat comprend notamment :

les températures ;
les précipitations ;
les vents ;
l'humidité ;
l'ensoleillement ;
les phénomènes météorologiques.

Ces éléments interagissent entre eux.
⸻
ÉVOLUTION

Le climat évolue selon :

les saisons ;
les régions ;
les catastrophes naturelles ;
les changements environnementaux.

Les variations restent cohérentes avec l'histoire du monde.
⸻
IMPACT

Le climat influence :

les écosystèmes ;
la faune ;
la flore ;
les ressources ;
les métiers ;
les déplacements ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée au climat devra :

respecter les équilibres naturels ;
produire des conséquences crédibles ;
évoluer avec le temps ;
enrichir les autres systèmes ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle le sentiment d'un climat vivant qui 
influence réellement le monde ?

Si la réponse est non, elle devra être repensée.
