GDB-010F --- Les Biomes

Version : 1.0
Statut : Officiel
Type : Monde Vivant
⸻


OBJECTIF

Définir les principes fondamentaux des biomes dans Chroniques.

Les biomes représentent les grands ensembles naturels du monde. Chacun 
possède son identité, ses espèces, ses ressources et ses contraintes 
écologiques.
⸻
PRINCIPE

Un biome est un environnement vivant façonné par le climat, le relief, 
les sols et les êtres vivants.

Chaque biome évolue naturellement et participe à la diversité du monde.
⸻
LES COMPOSANTES

Un biome comprend notamment :

un climat ;
une flore ;
une faune ;
des ressources ;
des paysages ;
des phénomènes naturels.

Chaque combinaison crée une identité unique.
⸻
ÉVOLUTION

Les biomes évoluent sous l'effet :

des saisons ;
des changements climatiques ;
des catastrophes naturelles ;
des activités humaines ;
des interactions écologiques.

Leur évolution reste progressive et cohérente.
⸻
IMPACT

Les biomes influencent :

l'exploration ;
les métiers ;
les ressources ;
les civilisations ;
les déplacements ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux biomes devra :

respecter les équilibres écologiques ;
offrir une identité forte ;
évoluer avec le monde ;
interagir avec les autres systèmes ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle à chaque biome une identité vivante 
plutôt qu'une simple variation visuelle ?

Si la réponse est non, elle devra être repensée.
