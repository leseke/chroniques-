GDB-010D --- La Flore

Version : 1.0
Statut : Officiel
Type : Monde Vivant
⸻


OBJECTIF

Définir les principes fondamentaux de la flore dans Chroniques.

La flore constitue l'une des bases du monde vivant. Elle façonne les 
paysages, nourrit les écosystèmes et influence directement les 
ressources, le climat et les activités des habitants.
⸻
PRINCIPE

Les végétaux existent indépendamment du joueur.

Ils germent, grandissent, se reproduisent, vieillissent et disparaissent 
selon des cycles naturels propres à chaque espèce.
⸻
DIVERSITÉ

La flore comprend notamment :

les arbres ;
les arbustes ;
les plantes ;
les fleurs ;
les mousses ;
les champignons ;
les cultures.

Chaque espèce possède ses besoins et son rôle écologique.
⸻
ÉVOLUTION

La flore évolue selon :

les saisons ;
le climat ;
la qualité des sols ;
les interactions avec la faune ;
les activités humaines.

Les paysages changent progressivement au fil du temps.
⸻
IMPACT

La flore influence :

les écosystèmes ;
les ressources ;
les métiers ;
l'agriculture ;
l'exploration ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée à la flore devra :

respecter les cycles naturels ;
favoriser la biodiversité ;
évoluer avec le monde ;
produire des conséquences crédibles ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle le sentiment d'une flore vivante plutôt 
que d'une simple collection de ressources ?

Si la réponse est non, elle devra être repensée.
⸻
