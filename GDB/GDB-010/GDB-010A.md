GDB-010A --- Le Monde Vivant

Version : 1.0
Statut : Officiel
Type : Monde Vivant
⸻


OBJECTIF

Définir les principes fondamentaux du monde vivant dans Chroniques.

Le monde n'est pas un décor. Il constitue un écosystème dynamique où 
chaque élément influence les autres et évolue au fil du temps.
⸻
PRINCIPE

Le monde existe indépendamment du joueur.

Les habitants, les animaux, les végétaux, le climat et les ressources 
suivent leurs propres cycles de vie.

Le joueur participe à cet équilibre sans en être le centre.
⸻
LES COMPOSANTES DU MONDE

Le monde vivant comprend notamment :

les écosystèmes ;
la faune ;
la flore ;
les climats ;
les ressources ;
les paysages ;
les sociétés.

Tous ces éléments interagissent continuellement.
⸻
ÉVOLUTION

Le monde évolue sous l'effet :

des saisons ;
des générations ;
des catastrophes naturelles ;
des découvertes ;
des activités humaines ;
des conséquences des actions.

Aucun territoire n'est totalement figé.
⸻
IMPACT

Le monde vivant influence :

l'exploration ;
les métiers ;
l'économie ;
les civilisations ;
les histoires émergentes.

Il constitue le socle de toutes les expériences vécues.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée au monde vivant devra :

respecter la cohérence écologique ;
favoriser les interactions naturelles ;
évoluer avec le temps ;
produire des conséquences crédibles ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle le sentiment d'un monde réellement vivant 
plutôt que d'un décor statique ?

Si la réponse est non, elle devra être repensée.
⸻
