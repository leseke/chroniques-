GDB-010H --- Les Ressources Naturelles

Version : 1.0
Statut : Officiel
Type : Monde Vivant
⸻


OBJECTIF

Définir les principes fondamentaux des ressources naturelles dans 
Chroniques.

Les ressources naturelles constituent la base matérielle du monde. Elles 
alimentent les écosystèmes, les métiers, les civilisations et l'économie 
tout en restant intégrées à leur environnement.
⸻
PRINCIPE

Une ressource naturelle existe indépendamment du joueur.

Elle apparaît, évolue, se renouvelle ou s'épuise selon les lois 
naturelles du monde et les actions des êtres vivants.
⸻
CATÉGORIES

Les ressources naturelles comprennent notamment :

l'eau ;
les minerais ;
les sols ;
le bois ;
les plantes sauvages ;
les animaux ;
les sources d'énergie naturelles.

Chaque ressource possède son propre cycle.
⸻
ÉVOLUTION

Les ressources évoluent selon :

le climat ;
les saisons ;
les écosystèmes ;
les catastrophes naturelles ;
les activités humaines.

Leur abondance n'est jamais définitivement acquise.
⸻
IMPACT

Les ressources influencent :

les métiers ;
les échanges ;
les biomes ;
les civilisations ;
les conflits ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux ressources naturelles devra :

respecter les équilibres écologiques ;
favoriser le renouvellement naturel lorsque cela est cohérent ;
produire des conséquences crédibles en cas de surexploitation ;
interagir avec les autres systèmes ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique traite-t-elle les ressources naturelles comme une 
partie vivante du monde plutôt que comme un stock infini ?

Si la réponse est non, elle devra être repensée.
⸻
