GDB-028B --- Les Valeurs Culturelles

Version : 1.0
Statut : Officiel
Type : Culture
⸻


OBJECTIF

Définir les principes fondamentaux des valeurs culturelles dans 
Chroniques.

Les valeurs culturelles représentent les principes, les idéaux et les 
normes qui orientent les comportements d'une société. Elles influencent 
les décisions individuelles et collectives ainsi que les relations entre 
les civilisations.
⸻
PRINCIPE

Les valeurs culturelles se construisent à partir de l'histoire, des 
traditions, des croyances et des expériences d'une population.

Elles évoluent progressivement sous l'effet des échanges, des crises, 
des innovations et des générations.
⸻
LES COMPOSANTES

Les valeurs culturelles comprennent généralement :

les principes moraux ;
les normes sociales ;
les idéaux collectifs ;
les traditions ;
les symboles ;
les références historiques ;
les comportements valorisés.
⸻
ÉVOLUTION

Les valeurs évoluent grâce :

aux échanges culturels ;
aux événements historiques ;
aux progrès des connaissances ;
aux transformations sociales ;
aux générations.
⸻
IMPACT

Les valeurs culturelles influencent :

l'identité ;
la diplomatie ;
la politique ;
l'économie ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux valeurs culturelles devra :

représenter leur diversité ;
permettre leur évolution ;
créer des interactions entre cultures ;
interagir avec les autres systèmes du monde ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle des valeurs culturelles un moteur des 
civilisations plutôt qu'un simple élément narratif ?

Si la réponse est non, elle devra être repensée.
