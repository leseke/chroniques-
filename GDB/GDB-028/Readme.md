# GDB-028

> Les éléments qui façonnent l'identité culturelle des peuples et des civilisations.

## Présentation

La série **GDB-028** définit les systèmes culturels de Chroniques. Elle décrit les valeurs, les traditions, les arts, le patrimoine et les échanges qui construisent l'identité des communautés au fil des générations.

## Objectif

Répondre à la question :

> Comment les cultures naissent-elles, évoluent-elles et se transmettent-elles ?

## Cette série couvre

- les fondements de la culture ;
- les valeurs culturelles ;
- les traditions ;
- les coutumes ;
- les arts ;
- le patrimoine culturel ;
- les échanges culturels ;
- les institutions culturelles ;
- la diversité culturelle ;
- la philosophie de la culture.
