GDB-003J --- Les Biomes

Version : 1.0
Statut : Officiel
Type : Architecture du Monde
⸻


OBJECTIF

Définir le rôle des biomes dans Chroniques.

Les biomes donnent au monde son identité écologique et influencent 
directement la manière de vivre, d'explorer et de survivre.
⸻
PRINCIPE

Un biome est un environnement naturel cohérent.

Il est défini par l'interaction entre le climat, le relief, les 
ressources et les êtres vivants.
⸻
IDENTITÉ D'UN BIOME

Chaque biome possède :

	⁃	un climat dominant ;
	⁃	une flore caractéristique ;
	⁃	une faune spécifique ;
	⁃	des ressources naturelles ;
	⁃	des paysages reconnaissables ;
	⁃	des contraintes et opportunités propres.
⸻
DIVERSITÉ

Deux biomes ne doivent jamais offrir exactement la même expérience.

Chaque biome influence les déplacements, les métiers, les rencontres et 
les stratégies du joueur.
⸻
ÉVOLUTION

Les biomes évoluent naturellement avec :

	⁃	les saisons ;
	⁃	les événements du monde ;
	⁃	les conséquences des activités humaines ;
	⁃	les phénomènes naturels.
⸻
INTERACTIONS

Les biomes influencent directement :

	⁃	l'économie ;
	⁃	les métiers ;
	⁃	l'exploration ;
	⁃	la biodiversité ;
	⁃	les implantations humaines.
⸻
RÈGLES DE CONCEPTION

Tout biome devra :

	1.	posséder une identité forte ;
	2.	être écologiquement cohérent ;
	3.	proposer des interactions uniques ;
	4.	enrichir les histoires émergentes ;
	5.	rester compatible avec les autres systèmes.
⸻
CRITÈRE DE VALIDATION

Ce biome est-il reconnaissable par son fonctionnement autant que par 
son apparence ?

Si la réponse est non, il devra être repensé.
