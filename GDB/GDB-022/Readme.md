# GDB-022

> La santé, les soins et le bien-être des habitants.

---

# Présentation

La série **GDB-022** décrit les systèmes liés à la santé dans Chroniques, depuis la prévention jusqu'aux traitements, en passant par les maladies et les établissements de soins.

---

# Objectif

La série GDB-022 répond à la question :

> **Comment les habitants préservent-ils leur santé et font-ils face aux maladies ?**

---

# Cette série couvre

- la santé ;
- les maladies ;
- la prévention ;
- les soins ;
- les médicaments ;
- les professionnels de santé ;
- les établissements médicaux ;
- le rétablissement ;
- le bien-être ;
- la philosophie de la santé.

# Relations

Cette série interagit avec les besoins des habitants, le vieillissement, l'économie et l'éducation.

---

# Principes

- privilégier la prévention ;
- rendre la santé dynamique ;
- intégrer les conséquences à long terme ;
- faire évoluer les connaissances médicales.
