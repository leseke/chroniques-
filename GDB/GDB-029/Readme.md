# GDB-029

> Les principes, disciplines et mécanismes scientifiques qui permettent aux civilisations de comprendre, d'expliquer et de faire progresser le monde.

## Présentation

La série **GDB-029** définit les fondements de la science dans Chroniques. Elle couvre les disciplines scientifiques, les méthodes de recherche, les découvertes, les innovations et les institutions qui participent au développement des connaissances.

## Objectif

Répondre à la question :

> Comment les connaissances scientifiques sont-elles produites, validées et utilisées dans Chroniques ?

## Cette série couvre

- les fondements de la science ;
- les disciplines scientifiques ;
- la recherche scientifique ;
- les méthodes scientifiques ;
- les découvertes scientifiques ;
- les innovations scientifiques ;
- les applications scientifiques ;
- les institutions scientifiques ;
- l'éthique scientifique ;
- la philosophie de la science.
