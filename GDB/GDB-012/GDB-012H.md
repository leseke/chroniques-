GDB-012H --- L'Innovation

Version : 1.0
Statut : Officiel
Type : Métiers & Activités
⸻


OBJECTIF

Définir les principes fondamentaux de l'innovation dans Chroniques.

L'innovation représente l'apparition de nouvelles idées, techniques, 
outils ou méthodes qui permettent aux individus et aux civilisations 
d'élargir progressivement leurs possibilités sans rompre la cohérence du 
monde.
⸻
PRINCIPE

Une innovation est une évolution, non une rupture arbitraire.

Elle résulte de l'observation, de l'expérimentation, de la recherche, 
des échanges et de la transmission entre générations.
⸻
LES SOURCES D'INNOVATION

Les innovations peuvent naître de :

	⁃	la recherche ;
	⁃	l'expérience ;
	⁃	les besoins des communautés ;
	⁃	les échanges entre civilisations ;
	⁃	les découvertes ;
	⁃	les erreurs ;
	⁃	la transmission des savoirs.

Chaque innovation répond à un contexte crédible.
⸻
ÉVOLUTION

Les innovations se diffusent progressivement.

Elles peuvent être adoptées, améliorées, adaptées ou abandonnées selon 
leur utilité et les choix des sociétés.
⸻
IMPACT

Les innovations influencent :

	⁃	les métiers ;
	⁃	les outils ;
	⁃	les productions ;
	⁃	les chaînes de production ;
	⁃	les civilisations ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée à l'innovation devra :

	1.	découler naturellement du monde ;
	2.	respecter la cohérence historique et technologique ;
	3.	créer de nouvelles possibilités plutôt que remplacer les anciennes ;
	4.	interagir avec plusieurs systèmes ;
	5.	enrichir durablement le gameplay.
⸻
CRITÈRE DE VALIDATION

Cette innovation enrichit-elle naturellement le monde plutôt que 
d'introduire artificiellement une nouvelle fonctionnalité ?

Si la réponse est non, elle devra être repensée.
⸻
