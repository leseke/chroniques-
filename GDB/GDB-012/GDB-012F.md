GDB-012F --- Les Chaînes de Production

Version : 1.0
Statut : Officiel
Type : Métiers & Activités
⸻


OBJECTIF

Définir les principes fondamentaux des chaînes de production dans 
Chroniques.

Les chaînes de production relient les ressources, les métiers, les 
outils et les productions afin de créer une économie vivante où chaque 
activité dépend des autres.
⸻
PRINCIPE

Aucune production importante n'existe isolément.

Chaque bien résulte d'une succession d'étapes impliquant différents 
savoir-faire, ressources et acteurs.

La coopération est encouragée sans être imposée.
⸻
COMPOSANTES

Une chaîne de production comprend notamment :

	⁃	l'extraction ou la récolte ;
	⁃	la transformation ;
	⁃	la fabrication ;
	⁃	le transport ;
	⁃	la distribution ;
	⁃	l'utilisation ;
	⁃	le recyclage ou la réutilisation.

Chaque étape possède sa propre valeur.
⸻
ÉVOLUTION

Les chaînes évoluent grâce :

	⁃	aux innovations ;
	⁃	aux nouveaux outils ;
	⁃	aux spécialisations ;
	⁃	aux échanges ;
	⁃	aux besoins des sociétés.

Elles deviennent plus efficaces sans perdre leur cohérence.
⸻
IMPACT

Les chaînes de production influencent :

	⁃	les métiers ;
	⁃	l'économie ;
	⁃	les échanges ;
	⁃	les communautés ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux chaînes de production devra :

	1.	relier plusieurs métiers ;
	2.	valoriser la coopération et la spécialisation ;
	3.	produire des conséquences crédibles ;
	4.	évoluer avec le monde ;
	5.	enrichir durablement le gameplay.
⸻
CRITÈRE DE VALIDATION

Cette chaîne de production crée-t-elle des interactions significatives 
entre les métiers plutôt qu'une simple succession de fabrications ?

Si la réponse est non, elle devra être repensée.
⸻
