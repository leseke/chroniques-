GDB-012D --- Les Outils

Version : 1.0
Statut : Officiel
Type : Métiers & Activités
⸻


OBJECTIF

Définir les principes fondamentaux des outils dans Chroniques.

Les outils permettent aux personnages d'interagir plus efficacement avec 
le monde. Ils prolongent les compétences, ouvrent de nouvelles 
possibilités et accompagnent l'évolution des métiers.
⸻
PRINCIPE

Un outil n'est pas une simple amélioration numérique.

Il modifie les possibilités d'action, la précision, la qualité ou 
l'efficacité d'une activité, tout en restant cohérent avec le monde.
⸻
DIVERSITÉ

Les outils peuvent être :

	⁃	artisanaux ;
	⁃	agricoles ;
	⁃	scientifiques ;
	⁃	artistiques ;
	⁃	commerciaux ;
	⁃	industriels ;
	⁃	spécialisés.

Chaque outil répond à un besoin concret.
⸻
ÉVOLUTION

Les outils évoluent grâce :

	⁃	aux innovations ;
	⁃	aux matériaux ;
	⁃	aux compétences ;
	⁃	aux échanges ;
	⁃	aux générations.

Ils accompagnent naturellement le développement des civilisations.
⸻
IMPACT

Les outils influencent :

	⁃	les métiers ;
	⁃	la qualité des réalisations ;
	⁃	la productivité ;
	⁃	les échanges ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux outils devra :

	1.	ouvrir de nouvelles possibilités d'action ;
	2.	respecter la cohérence technologique du monde ;
	3.	évoluer avec les connaissances ;
	4.	interagir avec plusieurs systèmes ;
	5.	enrichir le gameplay sans remplacer les compétences.
⸻
CRITÈRE DE VALIDATION

Cet outil enrichit-il réellement les possibilités du joueur plutôt que 
de simplement augmenter ses performances ?

Si la réponse est non, elle devra être repensée.
