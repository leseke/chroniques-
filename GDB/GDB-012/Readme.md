# GDB-012

> Les métiers, les savoir-faire et les activités professionnelles.

## Présentation

La série **GDB-012** décrit les métiers présents dans Chroniques, les activités qui leur sont associées ainsi que la transmission des connaissances professionnelles.

## Objectif

Répondre à la question :

> Comment les habitants travaillent-ils, produisent-ils et transmettent-ils leurs savoir-faire ?

## Cette série couvre

- les métiers ;
- les activités ;
- les spécialisations ;
- les outils ;
- les productions ;
- les chaînes de production ;
- la qualité ;
- l'innovation ;
- la transmission des savoir-faire ;
- la philosophie des métiers.
