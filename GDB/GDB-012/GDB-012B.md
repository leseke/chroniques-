GDB-012B --- Les Activités

Version : 1.0
Statut : Officiel
Type : Métiers & Activités
⸻


OBJECTIF

Définir les principes fondamentaux des activités dans Chroniques.

Les activités représentent l'ensemble des occupations qu'un personnage 
peut exercer au quotidien. Elles structurent la vie du monde et 
permettent au joueur d'exprimer librement son style de jeu.
⸻
PRINCIPE

Une activité est une action durable réalisée dans un but précis.

Elle peut être productive, sociale, créative, culturelle, scientifique 
ou simplement personnelle.

Chaque activité possède une valeur propre.
⸻
LES GRANDES CATÉGORIES

Les activités comprennent notamment :

	⁃	produire ;
	⁃	explorer ;
	⁃	apprendre ;
	⁃	enseigner ;
	⁃	commercer ;
	⁃	construire ;
	⁃	créer ;
	⁃	se divertir.

Aucune catégorie n'est obligatoire ni supérieure aux autres.
⸻
ÉVOLUTION

Les activités évoluent avec :

	⁃	les compétences ;
	⁃	les outils ;
	⁃	les connaissances ;
	⁃	les besoins des communautés ;
	⁃	les innovations.

Le joueur enrichit progressivement son quotidien.
⸻
IMPACT

Les activités influencent :

	⁃	les métiers ;
	⁃	les relations ;
	⁃	l'économie ;
	⁃	les projets ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute activité devra :

	1.	avoir une utilité crédible ;
	2.	pouvoir évoluer avec le temps ;
	3.	interagir avec plusieurs systèmes ;
	4.	respecter la liberté du joueur ;
	5.	enrichir le monde vivant.
⸻
CRITÈRE DE VALIDATION

Cette activité donne-t-elle au joueur une nouvelle manière de vivre 
dans le monde plutôt qu'une simple tâche à répéter ?

Si la réponse est non, elle devra être repensée.
