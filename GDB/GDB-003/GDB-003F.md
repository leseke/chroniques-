GDB-003F --- Les Points d'Intérêt

Version : 1.0
Statut : Officiel
Type : Architecture du Monde
⸻


OBJECTIF

Définir le rôle des points d'intérêt dans Chroniques.

Les points d'intérêt constituent les éléments remarquables qui ponctuent 
le monde et encouragent naturellement la curiosité.
⸻
PRINCIPE

Un point d'intérêt est un lieu mémorable.

Il attire le joueur non parce qu'une interface l'y oblige, mais parce 
qu'il suscite une envie de découverte.
⸻
IDENTITÉ

Chaque point d'intérêt possède au minimum :

	⁃	une identité visuelle ;
	⁃	une histoire ou une signification ;
	⁃	une ou plusieurs interactions possibles ;
	⁃	une raison crédible d'exister.
⸻
DIVERSITÉ

Les points d'intérêt peuvent être :

	⁃	naturels ;
	⁃	historiques ;
	⁃	culturels ;
	⁃	mystérieux ;
	⁃	économiques ;
	⁃	sociaux.

Ils ne doivent jamais se limiter à une récompense.
⸻
DÉCOUVERTE

Un point d'intérêt doit pouvoir être découvert :

	⁃	par l'exploration ;
	⁃	par une rumeur ;
	⁃	par un habitant ;
	⁃	par une ancienne carte ;
	⁃	par une conséquence du monde.
⸻
ÉVOLUTION

Certains points d'intérêt évoluent.

Ils peuvent apparaître, disparaître ou changer d'état au fil du temps et 
des événements.
⸻
RÈGLES DE CONCEPTION

Tout point d'intérêt devra :

	1.	susciter la curiosité ;
	2.	renforcer l'identité du monde ;
	3.	raconter une histoire ;
	4.	proposer une ou plusieurs interactions ;
	5.	rester cohérent avec son environnement.
⸻
CRITÈRE DE VALIDATION

Ce point d'intérêt donne-t-il envie d'être découvert avant même d'être 
exploité ?

Si la réponse est non, il devra être repensé.
⸻
