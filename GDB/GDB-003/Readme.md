# GDB-003

> L'architecture géographique du monde de Chroniques.

---

# Présentation

La série **GDB-003** définit l'organisation spatiale du monde de Chroniques.

Elle décrit la structure géographique utilisée par la simulation, depuis l'échelle globale jusqu'aux lieux individuels. Elle établit les différents niveaux d'organisation du monde ainsi que leurs relations.

Cette série constitue la référence officielle concernant la construction du monde.

---

# Objectif

La série GDB-003 répond à la question :

> **Comment le monde est-il organisé physiquement ?**

Elle définit une hiérarchie stable permettant de construire un univers cohérent, évolutif et facilement exploitable par les autres systèmes du jeu.

---

# Cette série couvre

La série documente notamment :

- la structure générale du monde ;
- les continents ;
- les régions ;
- les zones ;
- les lieux ;
- les points d'intérêt ;
- les réseaux de déplacement ;
- les frontières naturelles ;
- les différentes échelles du monde ;
- les biomes.

---

# Cette série ne couvre pas

Les éléments suivants sont documentés dans d'autres séries :

- les habitants ;
- les organisations ;
- l'économie ;
- les systèmes politiques ;
- les événements ;
- l'implémentation technique.

# Relations

Cette série s'appuie sur les principes définis dans **GDB-001**.

Elle fournit la structure spatiale utilisée par les séries consacrées aux personnages, aux organisations, à l'économie, aux infrastructures et à la simulation.

---

# Principes

Les documents de cette série doivent :

- maintenir une hiérarchie géographique cohérente ;
- éviter toute duplication entre les niveaux du monde ;
- garantir une organisation extensible ;
- rester indépendants des choix techniques ;
- servir de référence unique pour toute notion géographique.

---

# Évolution

La série pourra évoluer avec l'ajout de nouveaux types de territoires ou de nouvelles échelles de simulation.

Toute évolution devra préserver la cohérence de la hiérarchie géographique existante.
