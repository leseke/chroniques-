# GDB-021

> L'acquisition, la transmission et la préservation des connaissances.

---

# Présentation

La série **GDB-021** définit le fonctionnement de l'éducation dans Chroniques. Elle décrit les mécanismes permettant aux individus d'apprendre, d'enseigner, de transmettre et de conserver les connaissances au fil des générations.

---

# Objectif

La série GDB-021 répond à la question :

> **Comment les connaissances sont-elles acquises, transmises et préservées dans Chroniques ?**

---

# Cette série couvre

- l'éducation ;
- les enseignants ;
- les élèves ;
- les méthodes d'apprentissage ;
- les établissements d'enseignement ;
- les bibliothèques ;
- les archives ;
- la transmission des connaissances ;
- les savoirs collectifs ;
- la philosophie de l'éducation.

---

# Cette série ne couvre pas

Cette série ne décrit pas :

- les recherches scientifiques ;
- les innovations technologiques ;
- les métiers spécialisés ;
- les systèmes politiques.

Cette série alimente directement les séries consacrées aux sciences, aux technologies, à la culture et aux métiers.

---

# Principes

- apprendre tout au long de la vie ;
- préserver les connaissances ;
- transmettre les savoirs entre générations ;
- favoriser l'évolution intellectuelle du monde.
