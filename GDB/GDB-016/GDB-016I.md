GDB-016I --- Les Médiations

Version : 1.0
Statut : Officiel
Type : Diplomatie & Politique
⸻


OBJECTIF

Définir les principes fondamentaux des médiations dans Chroniques.

La médiation permet à un acteur neutre d'aider plusieurs parties à 
résoudre un différend sans recourir à la force. Elle constitue un outil 
essentiel pour préserver la stabilité et limiter l'escalade des 
conflits.
⸻
PRINCIPE

La médiation repose sur la neutralité.

Le médiateur facilite le dialogue, clarifie les intérêts des différentes 
parties et recherche des solutions acceptables pour tous, sans imposer 
sa décision.
⸻
LES FORMES DE MÉDIATION

Les médiations peuvent notamment concerner :

	⁃	les conflits territoriaux ;
	⁃	les différends commerciaux ;
	⁃	les alliances fragilisées ;
	⁃	les crises politiques ;
	⁃	les tensions culturelles ;
	⁃	les conflits religieux ;
	⁃	les rivalités entre organisations.

Chaque médiation possède ses propres enjeux.
⸻
ÉVOLUTION

Les médiations évoluent grâce :

	⁃	à la confiance accordée au médiateur ;
	⁃	aux concessions des parties ;
	⁃	aux événements du monde ;
	⁃	aux organisations diplomatiques ;
	⁃	aux générations.

Une médiation peut réussir, échouer ou être suspendue.
⸻
IMPACT

Les médiations influencent :

	⁃	les relations diplomatiques ;
	⁃	les alliances ;
	⁃	les conflits ;
	⁃	la stabilité des territoires ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux médiations devra :

	1.	préserver la neutralité du médiateur ;
	2.	offrir plusieurs issues crédibles ;
	3.	valoriser le dialogue plutôt que la contrainte ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle de la médiation une véritable alternative à 
l'affrontement plutôt qu'une simple option de dialogue ?

Si la réponse est non, elle devra être repensée.
