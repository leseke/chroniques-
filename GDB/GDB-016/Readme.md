# GDB-016

> La diplomatie, les relations entre entités et la résolution pacifique des conflits.

## Présentation

La série **GDB-016** définit les mécanismes diplomatiques de Chroniques. Elle décrit les relations entre individus, communautés, organisations et civilisations lorsqu'ils cherchent à défendre leurs intérêts par le dialogue, la coopération ou la négociation.

## Objectif

Répondre à la question :

> Comment les acteurs du monde interagissent-ils sans recourir à la force ?

## Cette série couvre

- la diplomatie ;
- les alliances ;
- les traités ;
- les négociations ;
- la réputation diplomatique ;
- les sanctions diplomatiques ;
- les organisations diplomatiques ;
- les sommets diplomatiques ;
- les médiations ;
- la philosophie de la diplomatie.
