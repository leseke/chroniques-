GDB-027B --- Les Sources d'Information

Version : 1.0
Statut : Officiel
Type : Information
⸻


OBJECTIF

Définir les principes fondamentaux des sources d'information dans 
Chroniques.

Les sources d'information sont les origines des connaissances utilisées 
par les individus, les organisations et les civilisations. Elles 
conditionnent la qualité, la fiabilité et la valeur des informations 
exploitées.
⸻
PRINCIPE

Une source d'information produit, conserve ou transmet des données.

Sa crédibilité dépend de son exactitude, de son indépendance, de son 
accessibilité, de sa mise à jour et de ses méthodes de collecte.
⸻
LES TYPES DE SOURCES

Les sources peuvent être :

orales ;
écrites ;
scientifiques ;
historiques ;
institutionnelles ;
numériques ;
empiriques.

Chaque type présente des avantages, des limites et des risques 
spécifiques.
⸻
ÉVOLUTION

Les sources évoluent grâce :

aux découvertes ;
aux innovations technologiques ;
aux échanges culturels ;
aux méthodes de vérification ;
aux générations.
⸻
IMPACT

Les sources d'information influencent :

la recherche ;
la politique ;
le commerce ;
la diplomatie ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux sources d'information devra :

représenter leur niveau de fiabilité ;
distinguer les différentes origines des informations ;
permettre leur évolution dans le temps ;
interagir avec les autres systèmes du monde ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle des sources d'information un élément 
stratégique de la connaissance plutôt qu'un simple point d'origine des 
données ?

Si la réponse est non, elle devra être repensée.
