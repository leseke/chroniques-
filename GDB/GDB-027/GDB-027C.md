GDB-027C --- La Collecte de l'Information

Version : 1.0
Statut : Officiel
Type : Information
⸻


OBJECTIF

Définir les principes fondamentaux de la collecte de l'information dans 
Chroniques.

La collecte de l'information regroupe les méthodes permettant d'obtenir 
des données utiles sur le monde, les populations, les ressources et les 
événements afin de soutenir la prise de décision.
⸻
PRINCIPE

La qualité d'une information dépend fortement de la manière dont elle 
est collectée.

Une collecte efficace privilégie la précision, la diversité des sources, 
la régularité des observations et la vérification des données.
⸻
LES MÉTHODES

La collecte peut être réalisée par :

l'observation ;
les enquêtes ;
les capteurs ;
les archives ;
les rapports ;
les témoignages ;
les systèmes automatisés.

Chaque méthode possède ses avantages et ses limites.
⸻
ÉVOLUTION

Les méthodes évoluent grâce :

aux progrès technologiques ;
aux nouvelles découvertes ;
aux besoins des civilisations ;
aux investissements ;
aux générations.
⸻
IMPACT

La collecte de l'information influence :

la recherche ;
la stratégie ;
le commerce ;
la sécurité ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée à la collecte devra :

représenter la qualité des données recueillies ;
distinguer les différentes méthodes de collecte ;
permettre l'amélioration des capacités d'observation ;
interagir avec les autres systèmes d'information ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle de la collecte de l'information une activité 
stratégique plutôt qu'un simple remplissage de données ?

Si la réponse est non, elle devra être repensée.
⸻
