# GDB-027

> Les mécanismes de collecte, de traitement, de conservation et de diffusion de l'information.

## Présentation

La série **GDB-027** décrit le cycle complet de l'information dans Chroniques. Elle définit la manière dont les données deviennent des connaissances exploitables par les habitants, les organisations et les civilisations.

## Objectif

Répondre à la question :

> Comment l'information est-elle produite, protégée et exploitée dans Chroniques ?

## Cette série couvre

- les fondements de l'information ;
- les sources d'information ;
- la collecte ;
- le traitement ;
- le stockage ;
- la diffusion ;
- la sécurité de l'information ;
- les flux stratégiques ;
- les centres d'information ;
- la philosophie de l'information.
