GDB-005G --- Les Marchés

Version : 1.0
Statut : Officiel
Type : Économie & Progression
⸻


OBJECTIF

Définir les principes fondamentaux des marchés dans Chroniques.

Les marchés organisent la rencontre entre les besoins, les ressources, 
les produits et les acteurs du monde.
⸻
PRINCIPE

Un marché est un lieu physique ou abstrait où s'effectuent des échanges.

Il reflète l'état réel de l'économie plutôt qu'un système figé.
⸻
FONCTIONNEMENT

Les marchés évoluent selon :

	⁃	l'offre ;
	⁃	la demande ;
	⁃	les ressources disponibles ;
	⁃	les saisons ;
	⁃	les événements ;
	⁃	les échanges entre communautés.
⸻
ÉCONOMIE VIVANTE

Les prix ne sont pas arbitraires.

Ils évoluent naturellement lorsque les conditions du monde changent.

Les pénuries, les surplus et les innovations modifient progressivement 
les marchés.
⸻
DIVERSITÉ

Chaque marché possède sa propre identité.

Selon son emplacement, sa culture et son histoire, il peut privilégier 
certains biens, métiers ou savoir-faire.
⸻
IMPACT

Les marchés influencent :

	⁃	les métiers ;
	⁃	les routes commerciales ;
	⁃	les projets des joueurs ;
	⁃	les communautés ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux marchés devra :

	1.	rester crédible ;
	2.	évoluer avec le monde ;
	3.	éviter les prix artificiellement figés ;
	4.	favoriser les interactions économiques ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle l'impression d'un marché vivant plutôt 
que d'une simple interface d'achat et de vente ?

Si la réponse est non, elle devra être repensée.
⸻
