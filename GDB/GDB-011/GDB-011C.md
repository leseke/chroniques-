GDB-011C --- Les Décisions

Version : 1.0
Statut : Officiel
Type : Interactions & Gameplay
⸻


OBJECTIF

Définir les principes fondamentaux des décisions dans Chroniques.

Les décisions représentent les choix conscients du joueur. Elles 
orientent son parcours et participent durablement à l'évolution du 
monde.
⸻
PRINCIPE

Une décision consiste à choisir une direction parmi plusieurs 
possibilités.

Chaque décision possède des conséquences qui peuvent être immédiates ou 
apparaître bien plus tard.
⸻
LES SOURCES DE DÉCISION

Les décisions peuvent concerner notamment :

	⁃	les relations ;
	⁃	les métiers ;
	⁃	les projets ;
	⁃	les ressources ;
	⁃	les déplacements ;
	⁃	les alliances ;
	⁃	les priorités de vie.

Aucun choix n'est présenté comme universellement meilleur.
⸻
ÉVOLUTION

Les décisions gagnent en profondeur grâce :

	⁃	aux connaissances acquises ;
	⁃	aux expériences vécues ;
	⁃	aux générations ;
	⁃	aux conséquences passées.

Le joueur apprend à mieux anticiper les effets de ses choix.
⸻
IMPACT

Les décisions influencent :

	⁃	les habitants ;
	⁃	les sociétés ;
	⁃	l'économie ;
	⁃	les écosystèmes ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux décisions devra :

	1.	proposer plusieurs options crédibles ;
	2.	produire des conséquences cohérentes ;
	3.	respecter la liberté du joueur ;
	4.	éviter les faux choix ;
	5.	enrichir durablement le monde.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle au joueur le sentiment que ses décisions 
façonnent réellement son aventure ?

Si la réponse est non, elle devra être repensée.
⸻
