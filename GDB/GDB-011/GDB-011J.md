GDB-011J --- La Philosophie du Gameplay

Version : 1.0
Statut : Officiel
Type : Interactions & Gameplay
⸻


OBJECTIF

Définir la philosophie générale du gameplay dans Chroniques.

Le gameplay représente la manière dont le joueur vit le monde. Il relie 
les mécaniques, les interactions et les conséquences pour créer une 
expérience cohérente, libre et émergente.
⸻
PRINCIPE

Le gameplay n'est pas une succession d'actions.

Il est un dialogue permanent entre le joueur et un monde vivant qui 
réagit, évolue et conserve la mémoire des événements.

Chaque décision ouvre de nouvelles possibilités.
⸻
LES PILIERS

Le gameplay repose notamment sur :

	⁃	les interactions ;
	⁃	les actions ;
	⁃	les décisions ;
	⁃	les conséquences ;
	⁃	les opportunités ;
	⁃	l'adaptation ;
	⁃	les systèmes connectés ;
	⁃	les émergences.

Ces éléments forment un ensemble indissociable.
⸻
UNE EXPÉRIENCE EN ÉVOLUTION

Le gameplay évolue continuellement grâce :

	⁃	au temps ;
	⁃	aux générations ;
	⁃	aux connaissances ;
	⁃	aux relations ;
	⁃	aux transformations du monde.

Chaque partie raconte une histoire différente.
⸻
IMPACT

Cette philosophie renforce :

	⁃	la liberté du joueur ;
	⁃	l'immersion ;
	⁃	les histoires émergentes ;
	⁃	la rejouabilité ;
	⁃	la cohérence globale de Chroniques.
⸻
RÈGLES DE CONCEPTION

Toute mécanique de gameplay devra :

	1.	favoriser les choix significatifs ;
	2.	produire des conséquences crédibles ;
	3.	interagir avec plusieurs systèmes ;
	4.	respecter la liberté du joueur ;
	5.	enrichir durablement le monde vivant.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle naissance à une expérience émergente et 
mémorable plutôt qu'à une simple succession d'objectifs ?

Si la réponse est non, elle devra être repensée.
