GDB-011D --- Les Conséquences

Version : 1.0
Statut : Officiel
Type : Interactions & Gameplay
⸻


OBJECTIF

Définir les principes fondamentaux des conséquences dans Chroniques.

Les conséquences donnent du poids aux actions et aux décisions. Elles 
assurent la continuité entre les événements passés, le présent et 
l'avenir du monde.
⸻
PRINCIPE

Toute action significative produit une conséquence.

Certaines sont immédiates, d'autres apparaissent progressivement ou 
plusieurs générations plus tard.

Les conséquences peuvent être visibles ou discrètes, mais elles 
participent toujours à l'évolution du monde.
⸻
LES FORMES DE CONSÉQUENCES

Les conséquences peuvent affecter :

	⁃	les relations ;
	⁃	les ressources ;
	⁃	les communautés ;
	⁃	les écosystèmes ;
	⁃	les institutions ;
	⁃	les générations futures ;
	⁃	la mémoire du monde.
⸻
ÉVOLUTION

Les conséquences interagissent entre elles.

Une décision mineure peut produire des effets majeurs lorsqu'elle se 
combine à d'autres événements au fil du temps.
⸻
IMPACT

Les conséquences renforcent :

	⁃	la crédibilité du monde ;
	⁃	l'immersion ;
	⁃	les histoires émergentes ;
	⁃	la responsabilité du joueur ;
	⁃	la rejouabilité.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux conséquences devra :

	1.	rester cohérente avec le monde ;
	2.	éviter les effets arbitraires ;
	3.	proposer des conséquences positives comme négatives ;
	4.	respecter le passage du temps ;
	5.	enrichir durablement l'expérience.
⸻
CRITÈRE DE VALIDATION

Cette mécanique donne-t-elle le sentiment que le monde se souvient 
réellement des actions du joueur ?

Si la réponse est non, elle devra être repensée.
