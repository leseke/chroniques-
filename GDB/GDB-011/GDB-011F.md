GDB-011F --- Les Adaptations

Version : 1.0
Statut : Officiel
Type : Interactions & Gameplay
⸻


OBJECTIF

Définir les principes fondamentaux de l'adaptation dans Chroniques.

L'adaptation représente la capacité du joueur, des habitants et des 
sociétés à modifier leurs comportements face à l'évolution du monde.
⸻
PRINCIPE

Le monde change continuellement.

S'adapter consiste à comprendre ces changements et à ajuster ses choix 
plutôt qu'à répéter indéfiniment les mêmes stratégies.
⸻
LES SOURCES D'ADAPTATION

L'adaptation peut être motivée par :

	⁃	les saisons ;
	⁃	les catastrophes naturelles ;
	⁃	les innovations ;
	⁃	les conflits ;
	⁃	les ressources disponibles ;
	⁃	les relations ;
	⁃	les générations.

Chaque évolution ouvre de nouvelles possibilités.
⸻
ÉVOLUTION

L'adaptation est un processus permanent.

Les individus, les communautés et les civilisations développent 
progressivement de nouvelles pratiques pour répondre aux transformations 
du monde.
⸻
IMPACT

L'adaptation influence :

	⁃	les métiers ;
	⁃	les stratégies ;
	⁃	les relations ;
	⁃	les économies ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée à l'adaptation devra :

	1.	récompenser la compréhension du monde ;
	2.	éviter une stratégie optimale permanente ;
	3.	produire des conséquences crédibles ;
	4.	favoriser plusieurs réponses possibles ;
	5.	renforcer le caractère vivant du monde.
⸻
CRITÈRE DE VALIDATION

Cette mécanique encourage-t-elle le joueur à s'adapter à un monde en 
évolution plutôt qu'à répéter une solution unique ?

Si la réponse est non, elle devra être repensée.
