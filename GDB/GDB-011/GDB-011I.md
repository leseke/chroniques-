GDB-011I --- Les Émergences

Version : 1.0
Statut : Officiel
Type : Interactions & Gameplay
⸻


OBJECTIF

Définir les principes fondamentaux des phénomènes émergents dans 
Chroniques.

Les émergences représentent les situations inédites créées par 
l'interaction naturelle des systèmes du jeu, sans avoir été scénarisées 
à l'avance.
⸻
PRINCIPE

Une émergence naît lorsque plusieurs mécaniques interagissent.

Elle ne résulte pas d'un script fixe mais des conséquences cumulées des 
actions, des décisions, du temps et des interactions entre les systèmes.
⸻
LES SOURCES D'ÉMERGENCE

Les phénomènes émergents peuvent provenir de :

	⁃	l'économie ;
	⁃	les relations ;
	⁃	les générations ;
	⁃	les écosystèmes ;
	⁃	les catastrophes naturelles ;
	⁃	les innovations ;
	⁃	les décisions du joueur.

Chaque combinaison peut produire une histoire unique.
⸻
ÉVOLUTION

Les émergences évoluent naturellement avec le monde.

Elles peuvent disparaître, se transformer ou donner naissance à de 
nouvelles opportunités.
⸻
IMPACT

Les émergences renforcent :

	⁃	la rejouabilité ;
	⁃	l'immersion ;
	⁃	les histoires émergentes ;
	⁃	la profondeur du gameplay ;
	⁃	le sentiment d'un monde vivant.
⸻
RÈGLES DE CONCEPTION

Toute mécanique devra :

	1.	favoriser les interactions entre systèmes ;
	2.	éviter les événements artificiels ;
	3.	produire des conséquences crédibles ;
	4.	laisser une part d'imprévu ;
	5.	enrichir durablement le monde.
⸻
CRITÈRE DE VALIDATION

Cette mécanique permet-elle à des situations inédites d'apparaître 
naturellement grâce aux interactions du monde ?

Si la réponse est non, elle devra être repensée.
