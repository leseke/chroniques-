# GDB-011

> Les interactions entre le joueur, les systèmes et le monde.

## Présentation

La série **GDB-011** définit l'ensemble des interactions possibles dans Chroniques. Elle décrit la manière dont le joueur agit sur le monde, comment le monde répond à ces actions et comment ces interactions produisent un gameplay émergent.

## Objectif

Répondre à la question :

> Comment les actions du joueur produisent-elles des conséquences cohérentes dans Chroniques ?

## Cette série couvre

- les interactions ;
- les actions ;
- les décisions ;
- les conséquences ;
- les opportunités ;
- les adaptations ;
- les boucles de gameplay ;
- les systèmes connectés ;
- les phénomènes émergents ;
- la philosophie du gameplay.
