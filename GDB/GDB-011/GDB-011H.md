GDB-011H --- Les Systèmes Connectés

Version : 1.0
Statut : Officiel
Type : Interactions & Gameplay
⸻


OBJECTIF

Définir les principes fondamentaux des systèmes connectés dans 
Chroniques.

Chaque mécanique du jeu doit interagir avec les autres afin de former un 
ensemble cohérent où une action peut produire des effets dans plusieurs 
domaines.
⸻
PRINCIPE

Aucun système n'existe isolément.

Les métiers, l'économie, les relations, les générations, les écosystèmes 
et les institutions s'influencent mutuellement.
⸻
INTERCONNEXIONS

Une interaction peut notamment affecter :

	⁃	les ressources ;
	⁃	les habitants ;
	⁃	les métiers ;
	⁃	les communautés ;
	⁃	les écosystèmes ;
	⁃	les traditions ;
	⁃	les opportunités.

Les conséquences se propagent naturellement entre les systèmes.
⸻
ÉVOLUTION

Les connexions évoluent avec :

	⁃	le temps ;
	⁃	les décisions ;
	⁃	les innovations ;
	⁃	les générations ;
	⁃	les changements du monde.

Le réseau d'interactions devient plus riche au fil de l'aventure.
⸻
IMPACT

Les systèmes connectés renforcent :

	⁃	l'immersion ;
	⁃	la cohérence ;
	⁃	les histoires émergentes ;
	⁃	la rejouabilité ;
	⁃	la profondeur du gameplay.
⸻
RÈGLES DE CONCEPTION

Toute mécanique devra :

	1.	interagir avec plusieurs systèmes ;
	2.	produire des conséquences crédibles ;
	3.	éviter les fonctionnalités isolées ;
	4.	enrichir le monde vivant ;
	5.	préserver la liberté du joueur.
⸻
CRITÈRE DE VALIDATION

Cette mécanique enrichit-elle plusieurs systèmes du jeu plutôt que de 
fonctionner indépendamment ?

Si la réponse est non, elle devra être repensée.
⸻
