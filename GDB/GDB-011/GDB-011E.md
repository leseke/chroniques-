GDB-011E --- Les Opportunités

Version : 1.0
Statut : Officiel
Type : Interactions & Gameplay
⸻


OBJECTIF

Définir les principes fondamentaux des opportunités dans Chroniques.

Les opportunités représentent les possibilités offertes au joueur par 
l'évolution naturelle du monde. Elles émergent des interactions entre 
les systèmes plutôt que d'être artificiellement déclenchées.
⸻
PRINCIPE

Une opportunité est une possibilité d'action créée par le contexte.

Elle résulte des événements, des relations, des découvertes, du temps ou 
des conséquences des décisions.

Aucune opportunité n'est garantie.
⸻
LES SOURCES D'OPPORTUNITÉS

Les opportunités peuvent naître :

	⁃	d'une rencontre ;
	⁃	d'un changement climatique ;
	⁃	d'une innovation ;
	⁃	d'un conflit ;
	⁃	d'une alliance ;
	⁃	d'une découverte ;
	⁃	d'un héritage.

Le monde génère continuellement de nouvelles possibilités.
⸻
ÉVOLUTION

Les opportunités apparaissent, évoluent puis disparaissent 
naturellement.

Le joueur choisit librement de les saisir, de les ignorer ou d'en créer 
de nouvelles.
⸻
IMPACT

Les opportunités influencent :

	⁃	les projets ;
	⁃	les métiers ;
	⁃	l'économie ;
	⁃	les relations ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux opportunités devra :

	1.	émerger naturellement du monde ;
	2.	respecter la liberté du joueur ;
	3.	produire des conséquences crédibles ;
	4.	favoriser plusieurs approches ;
	5.	enrichir durablement le gameplay.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle naître des opportunités de manière organique 
plutôt que par des déclencheurs artificiels ?

Si la réponse est non, elle devra être repensée.
