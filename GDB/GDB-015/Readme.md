# GDB-015

> Les conflits et leur résolution.

## Présentation

La série **GDB-015** définit les principes qui encadrent les combats et les autres formes de conflits dans Chroniques.

Elle couvre aussi bien les affrontements physiques que leurs conséquences et les moyens de les résoudre.

## Objectif

Répondre à la question :

> Comment gérer les conflits de manière cohérente avec la philosophie de Chroniques ?

## Cette série couvre

- le combat ;
- les armes ;
- les protections ;
- les tactiques ;
- les blessures ;
- la récupération ;
- les conséquences des combats ;
- les conflits ;
- les résolutions de conflits ;
- la philosophie du combat.
