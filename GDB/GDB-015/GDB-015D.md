GDB-015D --- Les Tactiques

Version : 1.0
Statut : Officiel
Type : Combat
⸻


OBJECTIF

Définir les principes fondamentaux des tactiques dans Chroniques.

Les tactiques représentent les choix stratégiques réalisés avant et 
pendant un affrontement. Elles permettent d'adapter les actions au 
terrain, aux adversaires et aux ressources disponibles.
⸻
PRINCIPE

La victoire dépend davantage des décisions que de la puissance brute.

Observer, anticiper, coopérer et exploiter l'environnement sont des 
éléments aussi importants que l'équipement.
⸻
LES DIMENSIONS TACTIQUES

Les tactiques peuvent notamment reposer sur :

	⁃	le positionnement ;
	⁃	le terrain ;
	⁃	la mobilité ;
	⁃	la coordination ;
	⁃	la gestion des ressources ;
	⁃	l'effet de surprise ;
	⁃	l'adaptation.

Chaque situation favorise une approche différente.
⸻
ÉVOLUTION

Les tactiques évoluent grâce :

	⁃	à l'expérience ;
	⁃	aux connaissances ;
	⁃	aux innovations ;
	⁃	aux entraînements ;
	⁃	aux générations.

Les combattants développent progressivement de nouvelles approches.
⸻
IMPACT

Les tactiques influencent :

	⁃	les combats ;
	⁃	les communautés ;
	⁃	les innovations ;
	⁃	les histoires émergentes ;
	⁃	les relations entre factions.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux tactiques devra :

	1.	récompenser l'analyse et l'anticipation ;
	2.	éviter une stratégie universelle ;
	3.	offrir plusieurs approches crédibles ;
	4.	interagir avec les autres systèmes ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique récompense-t-elle l'intelligence tactique plutôt que 
la simple supériorité numérique ou matérielle ?

Si la réponse est non, elle devra être repensée.
⸻
