GDB-015H --- Les Conflits

Version : 1.0
Statut : Officiel
Type : Combat
⸻


OBJECTIF

Définir les principes fondamentaux des conflits dans Chroniques.

Les conflits représentent les oppositions d'intérêts entre individus, 
groupes, communautés ou civilisations. Le combat n'est qu'une des 
nombreuses façons de les résoudre.
⸻
PRINCIPE

Un conflit naît d'une divergence.

Ses causes peuvent être économiques, politiques, culturelles, 
territoriales, religieuses, idéologiques ou personnelles. Chaque conflit 
suit sa propre évolution selon les décisions des acteurs concernés.
⸻
LES FORMES DE CONFLITS

Les conflits peuvent notamment être :

	⁃	personnels ;
	⁃	familiaux ;
	⁃	communautaires ;
	⁃	territoriaux ;
	⁃	économiques ;
	⁃	diplomatiques ;
	⁃	militaires.

Chaque forme possède ses propres mécanismes et conséquences.
⸻
ÉVOLUTION

Les conflits évoluent grâce :

	⁃	aux négociations ;
	⁃	aux alliances ;
	⁃	aux rivalités ;
	⁃	aux ressources ;
	⁃	aux événements du monde.

Ils peuvent s'intensifier, se stabiliser ou disparaître.
⸻
IMPACT

Les conflits influencent :

	⁃	les relations ;
	⁃	les communautés ;
	⁃	les territoires ;
	⁃	les économies ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux conflits devra :

	1.	privilégier plusieurs moyens de résolution ;
	2.	produire des conséquences durables ;
	3.	respecter la cohérence du monde ;
	4.	interagir avec tous les grands systèmes ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique transforme-t-elle chaque conflit en une situation 
dynamique pouvant évoluer de multiples façons plutôt qu'en une simple 
bataille à gagner ?

Si la réponse est non, elle devra être repensée.
