GDB-015A --- Le Combat

Version : 1.0
Statut : Officiel
Type : Combat
⸻


OBJECTIF

Définir les principes fondamentaux du combat dans Chroniques.

Le combat est un moyen de résoudre certains conflits, jamais une 
finalité. Il s'intègre au monde vivant et produit des conséquences 
durables sur les individus, les communautés et les territoires.
⸻
PRINCIPE

Combattre est un choix.

Chaque affrontement implique des risques, des coûts et des conséquences. 
La négociation, la fuite ou la coopération demeurent des alternatives 
crédibles lorsque la situation le permet.
⸻
LES FORMES DE COMBAT

Le combat peut notamment être :

	⁃	individuel ;
	⁃	collectif ;
	⁃	défensif ;
	⁃	offensif ;
	⁃	tactique ;
	⁃	improvisé ;
	⁃	organisé.

Chaque situation exige une approche adaptée.
⸻
ÉVOLUTION

Le combat évolue grâce :

	⁃	à l'expérience ;
	⁃	aux entraînements ;
	⁃	aux équipements ;
	⁃	aux connaissances tactiques ;
	⁃	aux innovations.

La maîtrise repose sur les choix du joueur plutôt que sur la répétition.
⸻
IMPACT

Le combat influence :

	⁃	les relations ;
	⁃	les communautés ;
	⁃	les ressources ;
	⁃	les territoires ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée au combat devra :

	1.	privilégier les choix tactiques ;
	2.	produire des conséquences crédibles ;
	3.	respecter la cohérence du monde ;
	4.	offrir plusieurs solutions à un conflit ;
	5.	renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle du combat une décision réfléchie plutôt 
qu'une activité systématique ?

Si la réponse est non, elle devra être repensée.
⸻
