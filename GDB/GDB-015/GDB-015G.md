GDB-015G --- Les Conséquences des Combats

Version : 1.0
Statut : Officiel
Type : Combat
⸻


OBJECTIF

Définir les principes fondamentaux des conséquences des combats dans 
Chroniques.

Les combats produisent des effets durables qui dépassent largement la 
victoire ou la défaite. Ils modifient les individus, les communautés, 
les territoires et l'histoire du monde.
⸻
PRINCIPE

Chaque combat laisse une trace.

Les conséquences peuvent être immédiates ou apparaître avec le temps. 
Elles dépendent des décisions prises avant, pendant et après 
l'affrontement.
⸻
LES TYPES DE CONSÉQUENCES

Les conséquences peuvent notamment être :

	⁃	humaines ;
	⁃	psychologiques ;
	⁃	sociales ;
	⁃	économiques ;
	⁃	politiques ;
	⁃	environnementales ;
	⁃	culturelles.

Chaque affrontement influence durablement le monde.
⸻
ÉVOLUTION

Les conséquences évoluent grâce :

	⁃	aux soins ;
	⁃	aux reconstructions ;
	⁃	aux décisions politiques ;
	⁃	aux relations entre communautés ;
	⁃	au passage des générations.

Certaines disparaissent, d'autres marquent durablement l'histoire.
⸻
IMPACT

Les conséquences influencent :

	⁃	les personnages ;
	⁃	les familles ;
	⁃	les communautés ;
	⁃	les territoires ;
	⁃	les civilisations ;
	⁃	les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux conséquences des combats devra :

	1.	produire des effets crédibles ;
	2.	éviter les affrontements sans impact ;
	3.	récompenser les choix réfléchis ;
	4.	interagir avec l'ensemble des systèmes du monde ;
	5.	renforcer le sentiment d'un univers vivant.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle du combat un événement ayant de véritables 
répercussions sur le monde plutôt qu'une simple succession de 
victoires et de défaites ?

Si la réponse est non, elle devra être repensée.
⸻
