# GDB-007

> La progression et la maîtrise du joueur.

---

# Présentation

La série **GDB-007** décrit la manière dont le joueur apprend, progresse et développe sa maîtrise au fil de la partie.

Elle définit les mécanismes qui permettent d'acquérir de nouvelles compétences, de relever des défis et de transformer l'expérience accumulée en véritable expertise.

---

# Objectif

La série GDB-007 répond à la question :

> **Comment le joueur apprend-il et devient-il meilleur dans Chroniques ?**

---

# Cette série couvre

- les compétences ;
- la maîtrise ;
- l'apprentissage ;
- l'expérience ;
- les erreurs ;
- les défis ;
- les réussites ;
- les échecs ;
- les styles de progression ;
- la philosophie de progression.

---

# Cette série ne couvre pas

Cette série ne décrit pas :

- les émotions recherchées ;
- les habitants ;
- l'économie ;
- les organisations ;
- les aspects techniques du moteur.

# Relations

Cette série complète GDB-006 en transformant les objectifs d'expérience en parcours de progression concrets.

Elle interagit avec toutes les séries décrivant les activités, les systèmes et les interactions du jeu.

---

# Principes

- récompenser la compréhension plutôt que la répétition ;
- transformer les erreurs en apprentissages ;
- proposer une progression durable ;
- laisser émerger différents styles de jeu ;
- valoriser la maîtrise plutôt que la simple accumulation.
