# GDB-019

> Les mécanismes économiques et commerciaux.

## Présentation

La série **GDB-019** décrit les échanges économiques de Chroniques. Elle couvre les ressources, les marchés, les entreprises et les mécanismes qui structurent l'économie du monde.

## Objectif

Répondre à la question :

> Comment fonctionne l'économie de Chroniques ?

## Cette série couvre

- l'économie ;
- les ressources économiques ;
- les monnaies ;
- les échanges commerciaux ;
- les marchés ;
- l'offre et la demande ;
- les prix ;
- les commerces ;
- les entreprises ;
- la philosophie de l'économie.
