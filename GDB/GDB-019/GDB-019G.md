GDB-019G --- Les Prix

Version : 1.0
Statut : Officiel
Type : Économie & Commerce
⸻


OBJECTIF

Définir les principes fondamentaux des prix dans Chroniques.

Les prix représentent la valeur d'échange des biens, des services et des 
ressources à un instant donné. Ils traduisent l'équilibre entre l'offre, 
la demande, les coûts de production et les conditions économiques du 
monde.
⸻
PRINCIPE

Un prix est une valeur dynamique.

Il évolue selon les coûts, la rareté, la concurrence, les taxes, les 
risques, les préférences des consommateurs et les événements affectant 
l'économie.
⸻
LES FACTEURS DE PRIX

Les prix sont notamment influencés par :

les coûts de production ;
l'offre et la demande ;
la qualité ;
la rareté ;
la concurrence ;
les taxes ;
les crises économiques.

Chaque facteur contribue à la formation du prix.
⸻
ÉVOLUTION

Les prix évoluent grâce :

aux innovations ;
aux variations de ressources ;
aux décisions politiques ;
aux échanges commerciaux ;
aux générations.

Ils peuvent augmenter, diminuer ou se stabiliser selon les conditions du 
marché.
⸻
IMPACT

Les prix influencent :

les consommateurs ;
les producteurs ;
les marchés ;
les investissements ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux prix devra :

refléter les réalités économiques du monde ;
varier de manière crédible ;
éviter les valeurs fixes injustifiées ;
interagir avec les autres systèmes ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle des prix une conséquence naturelle de 
l'économie plutôt qu'une simple valeur prédéfinie ?

Si la réponse est non, elle devra être repensée.
