GDB-019B --- Les Ressources Économiques

Version : 1.0 Statut : Officiel Type : Économie & Commerce
⸻
OBJECTIF

Définir les principes fondamentaux des ressources économiques dans 
Chroniques.

Les ressources économiques constituent la base de toute activité de 
production, de transformation et d'échange. Elles déterminent les 
capacités de développement des individus, des entreprises, des villes et 
des civilisations.
⸻
PRINCIPE

Une ressource économique possède une valeur parce qu'elle répond à un 
besoin.

Sa valeur dépend de sa disponibilité, de sa qualité, de sa difficulté 
d'obtention, de sa demande et des usages qui peuvent en être faits.
⸻
LES CATÉGORIES DE RESSOURCES

Les ressources économiques peuvent notamment être :

naturelles ;
agricoles ;
animales ;
minières ;
énergétiques ;
manufacturées ;
immatérielles.

Chaque ressource possède ses propres caractéristiques économiques.
⸻
ÉVOLUTION

Les ressources évoluent grâce :

à leur exploitation ;
aux découvertes ;
aux innovations ;
aux échanges ;
aux transformations du monde.

Leur abondance ou leur rareté peut évoluer avec le temps.
⸻
IMPACT

Les ressources influencent :

la production ;
les échanges commerciaux ;
les technologies ;
les conflits ;
les histoires émergentes.
⸻
RÈGLES DE CONCEPTION

Toute mécanique liée aux ressources économiques devra :

respecter les contraintes naturelles du monde ;
produire une rareté crédible ;
permettre plusieurs usages pour une même ressource ;
interagir avec les autres systèmes ;
renforcer l'immersion.
⸻
CRITÈRE DE VALIDATION

Cette mécanique fait-elle des ressources économiques le moteur naturel 
de l'économie plutôt qu'une simple liste de matériaux ?

Si la réponse est non, elle devra être repensée.
